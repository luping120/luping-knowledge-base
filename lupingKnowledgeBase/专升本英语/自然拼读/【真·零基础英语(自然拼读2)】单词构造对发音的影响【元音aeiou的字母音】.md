## 一、回顾：元音 a e i o u 的字母音

*Content-[00:00]*

核心规则：当单词结尾为不发音的 “e”，且从结尾向左数两格（即倒数第三个字母）是一个元音字母 (a, e, i, o, u) 时，该元音字母发它的字母音（即字母本身的读音）。

> 字母音：A 发 /eɪ/，E 发 /iː/，I 发 /aɪ/，O 发 /oʊ/，U 发 /juː/。

### 1. a 的字母音规则与示例

*Content-[00:45]*

- 构造：`C + a + C + e` (e 不发音，倒数第三格是 a)
- 示例单词：
  - cake → /keɪk/ (c 发 /k/，a 发字母音 /eɪ/，最后一个 e 不发音)
  - make → /meɪk/
  - sake → /seɪk/
  - made → /meɪd/
  - tape → /teɪp/
  - late → /leɪt/
  - shake → /ʃeɪk/
  - chase → /tʃeɪs/

### 2. e 的字母音规则与示例

*Content-[03:35]*

- 构造：`C + e + C + e`
- 示例单词：
  - these → /ðiːz/ (th 发 /ð/，e 发字母音 /iː/)
  - Pete → /piːt/
  - Eve → /iːv/

> 注意：符合该规则的 e 单词较少，常见的就是以上几个。

### 3. i 的字母音规则与示例

*Content-[05:10]*

- 构造：`C + i + C + e`
- 示例单词：
  - Mike → /maɪk/
  - bike → /baɪk/
  - bite → /baɪt/
  - side → /saɪd/
  - tide → /taɪd/
  - time → /taɪm/
  - vibe → /vaɪb/
  - wise → /waɪz/
  - dice → /daɪs/ (ce 结尾时 c 发 /s/)

### 4. o 的字母音规则与示例

*Content-[09:48]*

- 构造：`C + o + C + e`
- 示例单词：
  - robe → /roʊb/
  - mode → /moʊd/
  - coke → /koʊk/
  - smoke → /smoʊk/ (s 后清辅音不送气，读作 /d/)
  - dope → /doʊp/
  - hose → /hoʊz/
  - joke → /dʒoʊk/
  - nose → /noʊz/
  - poke → /poʊk/
  - close → /kloʊz/
  - stroke → /stroʊk/ (st 后的 t 浊化)

### 5. u 的字母音规则与示例

*Content-[12:57]*

- 构造：`C + u + C + e`
- 示例单词：
  - duke → /djuːk/
  - muse → /mjuːz/
  - cute → /kjuːt/
  - nuke → /njuːk/ (核武器)
  - puke → /pjuːk/ (呕吐)
  - tube → /tjuːb/ (管道；YouTube 的 tube)

## 二、扩展构造与更多示例

*Content-[02:56]*

规则不仅适用于四个字母的单词，五个或六个字母的单词同样适用。只要结尾是 e 不发音，且从结尾向左数两格是元音字母，该元音就发字母音。

- 例：`tribe` (5个字母，i 在倒数第三个位置 → /traɪb/)
- 例：`strike` (6个字母，i 在倒数第三个位置 → /straɪk/)
- 例：`spoke` (5个字母，o 发字母音 → /spoʊk/)

> 自然拼读法符合约 95% 的单词，存在少数例外，后续课程会补充。

## 三、重音与弱化对发音的影响

*Content-[13:18]*

当一个单词中有两个元音字母，其中一个已经按规则发字母音（重音在该音节），另一个元音则往往弱化。

- **规则**：一个单词只有一个主要重音。
- **弱化音**：弱化后的元音通常发 /ə/ 或 /ɪ/ （即 “非重读央元音” 或 “短 i 音”）。
- 示例：`museum` /mjuːˈziːəm/
  - 第一个 “u” 发字母音 /juː/，为重音。
  - 后面的 “e” 弱化为 /ɪ/，不读 /eɪ/。
  - 正确读音：mu-**se**-um，而非 *mu-**say**-um*。

## 四、常见缩略词与单词举例

*Content-[14:35]*

- `YouTube` = You + Tube (Tube 中的 u 发字母音 /juː/)
- `cute` 在口语中可形容人帅气/漂亮（相当于“正点”）
- `nuke` 是 nuclear weapon (核武器) 的缩写
- `puke` 意为呕吐 (I wanna puke. 我想吐。)

## 五、学习建议与总结

*Content-[07:13]*

- 自然拼读的规律是通过大量接触单词自然总结出来的，多看、多读、多练是最好的学习方法。
- 掌握规则后，阅读和拼写会更有目标，省时省力。
- 鼓励学习者自己观察和摸索规律，不必完全依赖教程。
