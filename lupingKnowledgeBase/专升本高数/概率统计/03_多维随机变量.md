## 3.1 二维随机变量

**内容待补充**：二维随机变量定义、联合分布函数、联合分布律、联合密度函数。

### 二维随机变量定义
设$X, Y$是定义在同一样本空间$\Omega$上的两个随机变量，则称$(X, Y)$为二维随机变量。

### 联合分布函数
定义：$F(x, y) = P\{X \leq x, Y \leq y\}$，$-\infty < x, y < \infty$

### 联合分布函数的性质
1. **单调性**：对每个变量单调不减
2. **有界性**：$0 \leq F(x, y) \leq 1$，$F(-\infty, y) = 0$，$F(x, -\infty) = 0$，$F(\infty, \infty) = 1$
3. **右连续性**：对每个变量右连续
4. **矩形区域概率**：$P\{x_1 < X \leq x_2, y_1 < Y \leq y_2\} = F(x_2, y_2) - F(x_2, y_1) - F(x_1, y_2) + F(x_1, y_1)$

### 二维离散型随机变量
联合分布律：$P\{X = x_i, Y = y_j\} = p_{ij}$，$i, j = 1, 2, \dots$
满足：$p_{ij} \geq 0$，$\sum_i \sum_j p_{ij} = 1$

### 二维连续型随机变量
联合密度函数：$f(x, y)$满足
$$F(x, y) = \int_{-\infty}^x \int_{-\infty}^y f(u, v) du dv$$
性质：$f(x, y) \geq 0$，$\int_{-\infty}^{\infty} \int_{-\infty}^{\infty} f(x, y) dx dy = 1$

## 3.2 边缘分布

**内容待补充**：边缘分布函数、边缘分布律、边缘密度函数。

### 边缘分布函数
1. $X$的边缘分布函数：$F_X(x) = P\{X \leq x\} = F(x, \infty)$
2. $Y$的边缘分布函数：$F_Y(y) = P\{Y \leq y\} = F(\infty, y)$

### 二维离散型情形
1. $X$的边缘分布律：$P\{X = x_i\} = p_{i\cdot} = \sum_j p_{ij}$
2. $Y$的边缘分布律：$P\{Y = y_j\} = p_{\cdot j} = \sum_i p_{ij}$

### 二维连续型情形
1. $X$的边缘密度函数：$f_X(x) = \int_{-\infty}^{\infty} f(x, y) dy$
2. $Y$的边缘密度函数：$f_Y(y) = \int_{-\infty}^{\infty} f(x, y) dx$

### 均匀分布
设$G$是平面有界区域，面积为$A$，若$(X, Y)$的联合密度为
$$f(x, y) = \begin{cases} \frac{1}{A}, & (x, y) \in G \\ 0, & \text{其他} \end{cases}$$
则称$(X, Y)$在$G$上服从均匀分布。

## 3.3 条件分布

**内容待补充**：离散型条件分布、连续型条件分布。

### 离散型条件分布
在$Y = y_j$条件下$X$的条件分布律：
$$P\{X = x_i | Y = y_j\} = \frac{p_{ij}}{p_{\cdot j}}，\quad p_{\cdot j} > 0$$
在$X = x_i$条件下$Y$的条件分布律：
$$P\{Y = y_j | X = x_i\} = \frac{p_{ij}}{p_{i\cdot}}，\quad p_{i\cdot} > 0$$

### 连续型条件分布
在$Y = y$条件下$X$的条件分布函数：
$$F_{X|Y}(x|y) = P\{X \leq x | Y = y\} = \int_{-\infty}^x \frac{f(u, y)}{f_Y(y)} du，\quad f_Y(y) > 0$$
在$Y = y$条件下$X$的条件密度函数：
$$f_{X|Y}(x|y) = \frac{f(x, y)}{f_Y(y)}，\quad f_Y(y) > 0$$
类似定义$f_{Y|X}(y|x)$。

### 条件分布的性质
1. 条件分布满足分布函数的性质
2. 对固定的条件值，条件密度是非负可积函数

## 3.4 随机变量的独立性

**内容待补充**：独立性定义、独立性判断、独立随机变量的性质。

### 独立性定义
随机变量$X, Y$独立 ⇔ 对任意实数$x, y$，有
$$F(x, y) = F_X(x)F_Y(y)$$

### 离散型情形
$X, Y$独立 ⇔ $p_{ij} = p_{i\cdot}p_{\cdot j}$ 对所有$i, j$成立

### 连续型情形
$X, Y$独立 ⇔ $f(x, y) = f_X(x)f_Y(y)$ 几乎处处成立

### 独立性的性质
1. 若$X, Y$独立，则$g(X)$与$h(Y)$也独立
2. 独立随机变量的函数仍独立
3. 零相关系数不一定独立（独立则相关系数为0，反之不一定）

### n维随机变量的独立性
随机变量$X_1, X_2, \dots, X_n$相互独立 ⇔
$$F(x_1, x_2, \dots, x_n) = F_{X_1}(x_1)F_{X_2}(x_2) \dots F_{X_n}(x_n)$$

## 3.5 多维随机变量函数的分布

**内容待补充**：和分布、极值分布、一般函数分布。

### 和的分布
设$(X, Y)$的联合密度为$f(x, y)$，则$Z = X + Y$的密度函数为
$$f_Z(z) = \int_{-\infty}^{\infty} f(x, z-x) dx = \int_{-\infty}^{\infty} f(z-y, y) dy$$

### 独立正态分布的和
若$X \sim N(\mu_1, \sigma_1^2)$，$Y \sim N(\mu_2, \sigma_2^2)$，且$X, Y$独立，则
$$X + Y \sim N(\mu_1 + \mu_2, \sigma_1^2 + \sigma_2^2)$$

### 商的分布
设$(X, Y)$的联合密度为$f(x, y)$，则$Z = \frac{X}{Y}$（$Y \neq 0$）的密度函数为
$$f_Z(z) = \int_{-\infty}^{\infty} |y| f(zy, y) dy$$

### 极值分布
设$X, Y$独立，分布函数分别为$F_X(x), F_Y(y)$，则
1. $M = \max(X, Y)$的分布函数：$F_M(z) = F_X(z)F_Y(z)$
2. $N = \min(X, Y)$的分布函数：$F_N(z) = 1 - [1 - F_X(z)][1 - F_Y(z)]$

### 一般函数分布（变换法）
设$(X, Y)$的联合密度为$f(x, y)$，$U = g_1(X, Y)$，$V = g_2(X, Y)$，若变换$(x, y) \to (u, v)$可逆，则$(U, V)$的联合密度为
$$f_{UV}(u, v) = f_{XY}(h_1(u, v), h_2(u, v))|J|$$
其中$J = \frac{\partial(x, y)}{\partial(u, v)}$为雅可比行列式。
