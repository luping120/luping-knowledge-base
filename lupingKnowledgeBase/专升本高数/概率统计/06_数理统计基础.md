## 6.1 样本与统计量

**内容待补充**：总体与样本、统计量定义、常见统计量。

### 总体与样本
1. **总体**：研究对象的全体，常用随机变量$X$表示
2. **样本**：从总体中随机抽取的$n$个个体$(X_1, X_2, \dots, X_n)$
3. **简单随机样本**：独立同分布于总体$X$

### 统计量定义
设$(X_1, X_2, \dots, X_n)$为总体$X$的样本，$T = T(X_1, X_2, \dots, X_n)$为样本的函数，且$T$中不含未知参数，则称$T$为统计量。

### 常见统计量
1. **样本均值**：$\overline{X} = \frac{1}{n}\sum_{i=1}^n X_i$
2. **样本方差**：$S^2 = \frac{1}{n-1}\sum_{i=1}^n (X_i - \overline{X})^2$
3. **样本标准差**：$S = \sqrt{S^2}$
4. **样本$k$阶原点矩**：$A_k = \frac{1}{n}\sum_{i=1}^n X_i^k$
5. **样本$k$阶中心矩**：$B_k = \frac{1}{n}\sum_{i=1}^n (X_i - \overline{X})^k$

### 统计量的性质
1. 统计量是随机变量
2. 统计量的分布称为抽样分布
3. 常用统计量有特定的抽样分布

## 6.2 抽样分布

**内容待补充**：$\chi^2$分布、$t$分布、$F$分布、正态总体下的抽样分布。

### $\chi^2$分布（卡方分布）
设$X_1, X_2, \dots, X_n$独立同分布于$N(0,1)$，则$\chi^2 = \sum_{i=1}^n X_i^2$服从自由度为$n$的$\chi^2$分布，记作$\chi^2 \sim \chi^2(n)$。

### $\chi^2$分布的性质
1. 可加性：若$\chi_1^2 \sim \chi^2(n_1)$，$\chi_2^2 \sim \chi^2(n_2)$，且独立，则$\chi_1^2 + \chi_2^2 \sim \chi^2(n_1 + n_2)$
2. 数学期望：$E(\chi^2(n)) = n$
3. 方差：$D(\chi^2(n)) = 2n$
4. 当$n$充分大时，$\chi^2(n)$近似$N(n, 2n)$

### $t$分布（学生氏分布）
设$X \sim N(0,1)$，$Y \sim \chi^2(n)$，且$X$与$Y$独立，则$T = \frac{X}{\sqrt{Y/n}}$服从自由度为$n$的$t$分布，记作$T \sim t(n)$。

### $t$分布的性质
1. 密度函数关于$t=0$对称
2. 当$n \to \infty$时，$t(n)$趋近于$N(0,1)$
3. $t_{1-\alpha}(n) = -t_{\alpha}(n)$

### $F$分布
设$U \sim \chi^2(n_1)$，$V \sim \chi^2(n_2)$，且$U$与$V$独立，则$F = \frac{U/n_1}{V/n_2}$服从自由度为$(n_1, n_2)$的$F$分布，记作$F \sim F(n_1, n_2)$。

### $F$分布的性质
1. 若$F \sim F(n_1, n_2)$，则$\frac{1}{F} \sim F(n_2, n_1)$
2. $F_{1-\alpha}(n_1, n_2) = \frac{1}{F_{\alpha}(n_2, n_1)}$

### 正态总体的抽样分布定理
设$X_1, X_2, \dots, X_n$为来自$N(\mu, \sigma^2)$的样本，$\overline{X}$为样本均值，$S^2$为样本方差，则
1. $\overline{X} \sim N\left(\mu, \frac{\sigma^2}{n}\right)$
2. $\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$
3. $\overline{X}$与$S^2$相互独立
4. $\frac{\overline{X} - \mu}{S/\sqrt{n}} \sim t(n-1)$

## 6.3 参数估计

**内容待补充**：点估计、矩估计法、最大似然估计、估计量的评价标准、区间估计。

### 点估计
用样本统计量的值来估计总体参数的值。

### 矩估计法
用样本矩估计总体矩：
1. 建立总体矩与参数的关系方程
2. 用样本矩代替总体矩
3. 解方程得到参数的矩估计量

### 最大似然估计
设总体$X$的密度（或分布律）为$f(x;\theta)$，$\theta \in \Theta$，样本$(X_1, X_2, \dots, X_n)$的观测值为$(x_1, x_2, \dots, x_n)$，则似然函数为
$$L(\theta) = \prod_{i=1}^n f(x_i;\theta)$$
最大似然估计$\hat{\theta}$是使$L(\theta)$达到最大的$\theta$值。

### 估计量的评价标准
1. **无偏性**：$E(\hat{\theta}) = \theta$
2. **有效性**：方差越小越有效
3. **一致性**：$\hat{\theta}_n \xrightarrow{P} \theta$（样本量$n \to \infty$）
4. **均方误差**：$MSE(\hat{\theta}) = E[(\hat{\theta} - \theta)^2]$

### 区间估计
寻找两个统计量$\hat{\theta}_1$和$\hat{\theta}_2$，使得
$$P\{\hat{\theta}_1 < \theta < \hat{\theta}_2\} = 1 - \alpha$$
其中$1-\alpha$称为置信水平，$(\hat{\theta}_1, \hat{\theta}_2)$称为$\theta$的置信区间。

### 单个正态总体参数的区间估计
1. **均值$\mu$的置信区间**（$\sigma^2$已知）：$\left(\overline{X} \pm z_{\alpha/2}\frac{\sigma}{\sqrt{n}}\right)$
2. **均值$\mu$的置信区间**（$\sigma^2$未知）：$\left(\overline{X} \pm t_{\alpha/2}(n-1)\frac{S}{\sqrt{n}}\right)$
3. **方差$\sigma^2$的置信区间**：$\left(\frac{(n-1)S^2}{\chi^2_{\alpha/2}(n-1)}, \frac{(n-1)S^2}{\chi^2_{1-\alpha/2}(n-1)}\right)$

## 6.4 假设检验

**内容待补充**：假设检验原理、两类错误、正态总体参数的假设检验。

### 假设检验的基本概念
1. **原假设$H_0$**：需要检验的假设
2. **备择假设$H_1$**：与原假设对立的假设
3. **检验统计量**：用于检验的样本函数
4. **拒绝域$W$**：拒绝原假设的样本值区域
5. **显著性水平$\alpha$**：犯第一类错误的概率

### 两类错误
1. **第一类错误（弃真）**：$H_0$为真时拒绝$H_0$，概率为$\alpha$
2. **第二类错误（取伪）**：$H_0$为假时接受$H_0$，概率为$\beta$

### 假设检验的步骤
1. 提出原假设$H_0$和备择假设$H_1$
2. 确定检验统计量及其分布
3. 给定显著性水平$\alpha$，确定拒绝域
4. 计算检验统计量的值
5. 作出决策：若统计量值落入拒绝域则拒绝$H_0$，否则接受$H_0$

### 正态总体均值的假设检验
1. **$\sigma^2$已知**（$Z$检验）：检验统计量$Z = \frac{\overline{X} - \mu_0}{\sigma/\sqrt{n}} \sim N(0,1)$
2. **$\sigma^2$未知**（$t$检验）：检验统计量$T = \frac{\overline{X} - \mu_0}{S/\sqrt{n}} \sim t(n-1)$

### 正态总体方差的假设检验
**$\chi^2$检验**：检验统计量$\chi^2 = \frac{(n-1)S^2}{\sigma_0^2} \sim \chi^2(n-1)$

### 两个正态总体的假设检验
1. **均值差的检验**（方差已知或未知但相等）
2. **方差比的检验**（$F$检验）
