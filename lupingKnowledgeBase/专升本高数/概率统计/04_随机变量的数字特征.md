## 4.1 数学期望

**内容待补充**：数学期望定义、性质、常见分布的期望。

### 离散型随机变量的数学期望
设$X$的分布律为$P\{X = x_k\} = p_k$，若级数$\sum_{k=1}^{\infty} x_k p_k$绝对收敛，则称
$$E(X) = \sum_{k=1}^{\infty} x_k p_k$$
为$X$的数学期望。

### 连续型随机变量的数学期望
设$X$的概率密度为$f(x)$，若积分$\int_{-\infty}^{\infty} x f(x) dx$绝对收敛，则称
$$E(X) = \int_{-\infty}^{\infty} x f(x) dx$$
为$X$的数学期望。

### 随机变量函数的数学期望
1. 离散型：$E[g(X)] = \sum_{k=1}^{\infty} g(x_k) p_k$
2. 连续型：$E[g(X)] = \int_{-\infty}^{\infty} g(x) f(x) dx$
3. 二维情形：$E[g(X, Y)] = \iint_{-\infty}^{\infty} g(x, y) f(x, y) dx dy$

### 数学期望的性质
1. **线性性**：$E(aX + bY + c) = aE(X) + bE(Y) + c$
2. **单调性**：若$X \leq Y$，则$E(X) \leq E(Y)$
3. **常数的期望**：$E(c) = c$
4. **独立乘积**：若$X, Y$独立，则$E(XY) = E(X)E(Y)$

### 常见分布的数学期望
1. 0-1分布：$E(X) = p$
2. 二项分布$B(n,p)$：$E(X) = np$
3. 泊松分布$P(\lambda)$：$E(X) = \lambda$
4. 均匀分布$U(a,b)$：$E(X) = \frac{a+b}{2}$
5. 指数分布$Exp(\lambda)$：$E(X) = \frac{1}{\lambda}$
6. 正态分布$N(\mu, \sigma^2)$：$E(X) = \mu$

## 4.2 方差

**内容待补充**：方差定义、性质、常见分布的方差。

### 方差定义
设$X$为随机变量，若$E[(X - E(X))^2]$存在，则称
$$D(X) = E[(X - E(X))^2]$$
为$X$的方差，称$\sqrt{D(X)}$为标准差。

### 方差的计算公式
$$D(X) = E(X^2) - [E(X)]^2$$

### 方差的性质
1. **非负性**：$D(X) \geq 0$，$D(X)=0$ ⇔ $X$以概率1取常数值
2. **常数方差**：$D(c) = 0$
3. **齐次性**：$D(aX) = a^2 D(X)$
4. **平移不变性**：$D(X + b) = D(X)$
5. **和的方差**：$D(X+Y) = D(X) + D(Y) + 2Cov(X, Y)$
6. 若$X, Y$独立，则$D(X+Y) = D(X) + D(Y)$

### 常见分布的方差
1. 0-1分布：$D(X) = p(1-p)$
2. 二项分布$B(n,p)$：$D(X) = np(1-p)$
3. 泊松分布$P(\lambda)$：$D(X) = \lambda$
4. 均匀分布$U(a,b)$：$D(X) = \frac{(b-a)^2}{12}$
5. 指数分布$Exp(\lambda)$：$D(X) = \frac{1}{\lambda^2}$
6. 正态分布$N(\mu, \sigma^2)$：$D(X) = \sigma^2$

### 切比雪夫不等式
设$X$的数学期望$E(X)$和方差$D(X)$存在，则对任意$\varepsilon > 0$有
$$P\{|X - E(X)| \geq \varepsilon\} \leq \frac{D(X)}{\varepsilon^2}$$

## 4.3 协方差与相关系数

**内容待补充**：协方差定义、相关系数定义、性质。

### 协方差定义
设$(X, Y)$为二维随机变量，若$E[(X - E(X))(Y - E(Y))]$存在，则称
$$Cov(X, Y) = E[(X - E(X))(Y - E(Y))]$$
为$X$与$Y$的协方差。

### 协方差计算公式
$$Cov(X, Y) = E(XY) - E(X)E(Y)$$

### 协方差的性质
1. **对称性**：$Cov(X, Y) = Cov(Y, X)$
2. **线性性**：$Cov(aX+b, cY+d) = ac Cov(X, Y)$
3. **分配律**：$Cov(X_1+X_2, Y) = Cov(X_1, Y) + Cov(X_2, Y)$
4. **与方差关系**：$Cov(X, X) = D(X)$
5. **独立则协方差为0**：若$X, Y$独立，则$Cov(X, Y) = 0$

### 相关系数定义
设$D(X) > 0$，$D(Y) > 0$，则称
$$\rho_{XY} = \frac{Cov(X, Y)}{\sqrt{D(X)}\sqrt{D(Y)}}$$
为$X$与$Y$的相关系数。

### 相关系数的性质
1. **有界性**：$|\rho_{XY}| \leq 1$
2. **线性关系**：$|\rho_{XY}| = 1$ ⇔ 存在常数$a, b$使$P\{Y = aX + b\} = 1$
3. **不相关性**：若$\rho_{XY} = 0$，称$X, Y$不相关
4. **独立与不相关**：独立 ⇒ 不相关，反之不一定

### 协方差矩阵
对$n$维随机向量$(X_1, X_2, \dots, X_n)$，协方差矩阵为
$$\Sigma = (Cov(X_i, X_j))_{n \times n}$$

## 4.4 矩与协方差矩阵

**内容待补充**：原点矩、中心矩、混合矩。

### 原点矩
设$X$为随机变量，$k$为正整数，若$E(X^k)$存在，则称$\nu_k = E(X^k)$为$X$的$k$阶原点矩。

### 中心矩
设$X$为随机变量，$k$为正整数，若$E[(X - E(X))^k]$存在，则称$\mu_k = E[(X - E(X))^k]$为$X$的$k$阶中心矩。
- 一阶中心矩：$\mu_1 = 0$
- 二阶中心矩：$\mu_2 = D(X)$

### 混合矩
设$(X, Y)$为二维随机变量，若$E(X^k Y^l)$存在，则称$\nu_{kl} = E(X^k Y^l)$为$k+l$阶混合原点矩。
若$E[(X - E(X))^k (Y - E(Y))^l]$存在，则称$\mu_{kl} = E[(X - E(X))^k (Y - E(Y))^l]$为$k+l$阶混合中心矩。

### 偏度与峰度
1. **偏度**：$\gamma_1 = \frac{\mu_3}{\sigma^3}$，描述分布对称性
   - $\gamma_1 > 0$：右偏（正偏）
   - $\gamma_1 < 0$：左偏（负偏）
   - $\gamma_1 = 0$：对称分布
2. **峰度**：$\gamma_2 = \frac{\mu_4}{\sigma^4} - 3$，描述分布尖锐程度
   - $\gamma_2 > 0$：比正态分布尖锐
   - $\gamma_2 < 0$：比正态分布平缓
   - $\gamma_2 = 0$：正态分布

### 协方差矩阵的性质
1. 对称性：$\Sigma^T = \Sigma$
2. 半正定性：对任意实向量$\vec{a}$，有$\vec{a}^T \Sigma \vec{a} \geq 0$
3. 对角元素为方差：$\Sigma_{ii} = D(X_i)$
4. 非对角元素为协方差：$\Sigma_{ij} = Cov(X_i, X_j)$

### 基础题
1. **期望计算**：设随机变量$X$的概率密度为$f(x) = \begin{cases} 2x, & 0 < x < 1 \\ 0, & \text{其他} \end{cases}$，求$E(X)$和$E(X^2)$

   X\backslash Y & 0 & 1 \\
