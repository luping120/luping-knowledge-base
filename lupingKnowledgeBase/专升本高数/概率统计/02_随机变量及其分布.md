## 2.1 随机变量的概念

**内容待补充**：随机变量定义、分布函数定义与性质。

### 随机变量定义
设随机试验的样本空间为$\Omega$，如果对于每个$\omega \in \Omega$，都有唯一实数$X(\omega)$与之对应，则称$X = X(\omega)$为随机变量。

### 分类
1. **离散型随机变量**：取值有限或可列无限
2. **连续型随机变量**：取值充满一个区间
3. **混合型随机变量**：既有离散部分又有连续部分

### 分布函数
定义：$F(x) = P\{X \leq x\}$，$-\infty < x < \infty$

### 分布函数的性质
1. **单调不减**：$x_1 < x_2 \Rightarrow F(x_1) \leq F(x_2)$
2. **有界性**：$0 \leq F(x) \leq 1$，$\lim_{x \to -\infty} F(x) = 0$，$\lim_{x \to \infty} F(x) = 1$
3. **右连续性**：$\lim_{x \to x_0^+} F(x) = F(x_0)$
4. **概率计算**：$P\{a < X \leq b\} = F(b) - F(a)$

## 2.2 离散型随机变量

**内容待补充**：概率分布列、常见离散分布。

### 概率分布列
离散型随机变量$X$的所有可能取值$x_k$及对应的概率$p_k$：
$$\begin{array}{c|ccccc}
X & x_1 & x_2 & \cdots & x_k & \cdots \\
\hline
P & p_1 & p_2 & \cdots & p_k & \cdots
\end{array}$$
其中$p_k \geq 0$，$\sum_k p_k = 1$

### 常见离散分布
1. **0-1分布**（伯努利分布）：$P\{X=1\} = p$，$P\{X=0\} = 1-p$
2. **二项分布**$B(n,p)$：$P\{X=k\} = C_n^k p^k (1-p)^{n-k}$，$k=0,1,\dots,n$
3. **泊松分布**$P(\lambda)$：$P\{X=k\} = \frac{\lambda^k e^{-\lambda}}{k!}$，$k=0,1,2,\dots$
4. **几何分布**：$P\{X=k\} = (1-p)^{k-1}p$，$k=1,2,\dots$
5. **超几何分布**：$P\{X=k\} = \frac{C_M^k C_{N-M}^{n-k}}{C_N^n}$，$k=0,1,\dots,\min(n,M)$

### 分布函数
离散型随机变量的分布函数为阶梯函数：
$$F(x) = P\{X \leq x\} = \sum_{x_k \leq x} p_k$$

## 2.3 连续型随机变量

**内容待补充**：概率密度函数、常见连续分布。

### 概率密度函数
连续型随机变量$X$的分布函数$F(x)$可表示为
$$F(x) = \int_{-\infty}^x f(t) dt$$
其中$f(x)$称为概率密度函数。

### 概率密度函数的性质
1. **非负性**：$f(x) \geq 0$
2. **归一性**：$\int_{-\infty}^{\infty} f(x) dx = 1$
3. **概率计算**：$P\{a < X < b\} = \int_a^b f(x) dx$
4. **单点概率**：$P\{X = x_0\} = 0$

### 常见连续分布
1. **均匀分布**$U(a,b)$：$f(x) = \begin{cases} \frac{1}{b-a}, & a < x < b \\ 0, & \text{其他} \end{cases}$
2. **指数分布**$Exp(\lambda)$：$f(x) = \begin{cases} \lambda e^{-\lambda x}, & x > 0 \\ 0, & x \leq 0 \end{cases}$，$\lambda > 0$
3. **正态分布**$N(\mu, \sigma^2)$：$f(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$，$-\infty < x < \infty$
4. **标准正态分布**：$\mu=0$，$\sigma=1$，记作$N(0,1)$

### 正态分布的性质
1. 对称性：关于$x=\mu$对称
2. 拐点：$x = \mu \pm \sigma$
3. $3\sigma$原则：$P\{|X-\mu| < \sigma\} \approx 0.6827$，$P\{|X-\mu| < 2\sigma\} \approx 0.9545$，$P\{|X-\mu| < 3\sigma\} \approx 0.9973$

## 2.4 随机变量函数的分布

**内容待补充**：离散型随机变量函数的分布、连续型随机变量函数的分布。

### 离散型情形
设$X$为离散型随机变量，分布列为$P\{X=x_k\} = p_k$，$Y = g(X)$，则$Y$的分布列为
$$P\{Y=y_j\} = \sum_{g(x_k)=y_j} p_k$$

### 连续型情形（公式法）
设$X$为连续型随机变量，密度函数为$f_X(x)$，$y = g(x)$严格单调可导，则$Y = g(X)$的密度函数为
$$f_Y(y) = f_X[h(y)]|h'(y)|$$
其中$x = h(y)$是$y = g(x)$的反函数。

### 连续型情形（分布函数法）
1. 求$Y$的分布函数：$F_Y(y) = P\{Y \leq y\} = P\{g(X) \leq y\}$
2. 对$F_Y(y)$求导得密度函数：$f_Y(y) = F_Y'(y)$

### 常见变换
1. **线性变换**：$Y = aX + b$
2. **平方变换**：$Y = X^2$
3. **指数变换**：$Y = e^X$
4. **正态分布的线性变换**：若$X \sim N(\mu, \sigma^2)$，则$aX+b \sim N(a\mu+b, a^2\sigma^2)$
