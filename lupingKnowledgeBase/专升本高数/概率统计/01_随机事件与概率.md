## 1.1 随机事件

**内容待补充**：随机现象、样本空间、随机事件、事件关系与运算。

### 随机现象
- **确定性现象**：结果必然发生
- **随机现象**：结果不确定，但有统计规律

### 样本空间
随机试验所有可能结果的集合，记作$\Omega$。

### 随机事件
样本空间的子集，常用大写字母$A, B, C, \dots$表示。

### 事件的关系与运算
1. **包含**：$A \subset B$（A发生则B发生）
2. **相等**：$A = B$
3. **和事件**：$A \cup B$（至少一个发生）
4. **积事件**：$A \cap B$（同时发生）
5. **差事件**：$A - B$（A发生但B不发生）
6. **互斥事件**：$A \cap B = \varnothing$（不能同时发生）
7. **对立事件**：$A \cup \overline{A} = \Omega$，$A \cap \overline{A} = \varnothing$

### 运算律
1. **交换律**：$A \cup B = B \cup A$，$A \cap B = B \cap A$
2. **结合律**：$(A \cup B) \cup C = A \cup (B \cup C)$
3. **分配律**：$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$
4. **德摩根律**：$\overline{A \cup B} = \overline{A} \cap \overline{B}$，$\overline{A \cap B} = \overline{A} \cup \overline{B}$

## 1.2 概率的定义与性质

**内容待补充**：概率的古典定义、几何定义、统计定义、公理化定义。

### 概率的古典定义
设随机试验的样本空间包含有限个等可能基本事件，事件$A$包含$m$个基本事件，则
$$P(A) = \frac{m}{n}$$
其中$n$为基本事件总数。

### 概率的几何定义
设样本空间$\Omega$为可度量的几何区域，事件$A$为$\Omega$的子区域，则
$$P(A) = \frac{\text{区域A的度量}}{\text{区域Ω的度量}}$$

### 概率的统计定义
在$n$次重复试验中，事件$A$发生$m$次，则频率$\frac{m}{n}$的稳定值称为事件$A$的概率。

### 概率的公理化定义（柯尔莫哥洛夫）
概率$P$是定义在事件域$\mathcal{F}$上的实值函数，满足：
1. **非负性**：$P(A) \geq 0$
2. **规范性**：$P(\Omega) = 1$
3. **可列可加性**：对两两互斥事件$A_1, A_2, \dots$，有$P\left(\bigcup_{i=1}^{\infty} A_i\right) = \sum_{i=1}^{\infty} P(A_i)$

### 概率的性质
1. $P(\varnothing) = 0$
2. $P(\overline{A}) = 1 - P(A)$
3. $P(A - B) = P(A) - P(AB)$
4. **加法公式**：$P(A \cup B) = P(A) + P(B) - P(AB)$
5. **一般加法公式**：$P\left(\bigcup_{i=1}^n A_i\right) = \sum_{i=1}^n P(A_i) - \sum_{1 \leq i < j \leq n} P(A_i A_j) + \cdots + (-1)^{n-1} P(A_1 A_2 \dots A_n)$

## 1.3 条件概率与独立性

**内容待补充**：条件概率定义、乘法公式、全概率公式、贝叶斯公式、事件独立性。

### 条件概率定义
设$P(B) > 0$，在事件$B$发生的条件下事件$A$发生的概率为
$$P(A|B) = \frac{P(AB)}{P(B)}$$

### 乘法公式
1. $P(AB) = P(B)P(A|B)$，$P(B) > 0$
2. $P(AB) = P(A)P(B|A)$，$P(A) > 0$
3. **推广**：$P(A_1 A_2 \dots A_n) = P(A_1)P(A_2|A_1)P(A_3|A_1A_2) \dots P(A_n|A_1A_2 \dots A_{n-1})$

### 全概率公式
设$B_1, B_2, \dots, B_n$为样本空间$\Omega$的一个划分，且$P(B_i) > 0$，则对任意事件$A$有
$$P(A) = \sum_{i=1}^n P(B_i)P(A|B_i)$$

### 贝叶斯公式
在$P(A) > 0$，$P(B_i) > 0$的条件下
$$P(B_i|A) = \frac{P(B_i)P(A|B_i)}{\sum_{j=1}^n P(B_j)P(A|B_j)}$$

### 事件的独立性
1. **两个事件独立**：$P(AB) = P(A)P(B)$
2. **等价条件**：$P(A|B) = P(A)$（当$P(B) > 0$）
3. **多个事件独立**：对任意$k$个事件，积事件的概率等于各事件概率的乘积

### 独立性的性质
1. 若$A$与$B$独立，则$A$与$\overline{B}$，$\overline{A}$与$B$，$\overline{A}$与$\overline{B}$也独立
2. 独立与互斥不同：互斥则$P(AB) = 0$，独立则$P(AB) = P(A)P(B)$
3. 零概率事件与任意事件独立

## 1.4 伯努利试验

**内容待补充**：伯努利试验定义、二项概率公式。

### 伯努利试验定义
满足以下条件的随机试验：
1. 每次试验只有两个可能结果：成功（A）或失败（$\overline{A}$）
2. 每次试验成功的概率相同：$P(A) = p$
3. 试验之间相互独立

### n重伯努利试验
将伯努利试验独立重复$n$次。

### 二项概率公式
在$n$重伯努利试验中，事件$A$恰好发生$k$次的概率为
$$P_n(k) = C_n^k p^k (1-p)^{n-k}, \quad k = 0, 1, 2, \dots, n$$
其中$C_n^k = \frac{n!}{k!(n-k)!}$为组合数。

### 二项分布的性质
1. $\sum_{k=0}^n P_n(k) = 1$
2. 最可能次数：使$P_n(k)$最大的$k$值
3. 当$n$很大，$p$很小时，可用泊松分布近似

### 基础题
1. **事件运算**：设$A, B, C$为三个事件，用$A, B, C$表示下列事件：
   (1) $A$发生但$B, C$都不发生
   (2) 恰好有两个事件发生
   (3) 至少有一个事件发生

2. **概率计算**：从1,2,3,4,5中任取两个数，求：
   (1) 和为偶数的概率
   (2) 积为偶数的概率

3. **条件概率**：已知$P(A) = 0.5$，$P(B) = 0.6$，$P(B|A) = 0.8$，求$P(A|B)$

   (1) 取到白球的概率
