## 分母为互异的一次因式

这是最基础的形式。如果分母可以分解为互不相同的线性因子，则每一项的分子都是常数。

| 裂项形式 | 原式 | 裂项后表达式 | 说明 |
| ------ |------ |------ |------ |
| 两个因式 | $\frac{P(x)}{(x-a)(x-b)}$ | $\frac{A}{x-a} + \frac{B}{x-b}$ | 分子次数需低于分母 |
| 三个因式 | $\frac{P(x)}{(x-a)(x-b)(x-c)}$ | $\frac{A}{x-a} + \frac{B}{x-b} + \frac{C}{x-c}$ | 依此类推 |

## 分母含有重复的一次因式

如果分母中有 $(x-a)^k$ 这样的项，必须为每一个次幂都设立一个待定系数。

| 裂项形式 | 原式 | 裂项后表达式 | 说明 |
| ------ |------ |------ |------ |
| 平方项 | $\frac{P(x)}{(x-a)^2}$ | $\frac{A}{x-a} + \frac{B}{(x-a)^2}$ | 需包含所有次幂 |
| 立方项 | $\frac{P(x)}{(x-a)^3}$ | $\frac{A}{x-a} + \frac{B}{(x-a)^2} + \frac{C}{(x-a)^3}$ | 需包含所有次幂 |
| 混合形式 | $\frac{P(x)}{(x-a)(x-b)^2}$ | $\frac{A}{x-a} + \frac{B}{x-b} + \frac{C}{(x-b)^2}$ | 分别处理不同因式 |

## 分母含有不可约的二次因式

如果分母含有 $ax^2+bx+c$（判别式 $\Delta < 0$），其对应的分子必须是**一次多项式** $Ax+B$。

| 裂项形式   | 原式                          | 裂项后表达式                               | 说明            |
| ------ | --------------------------- | ------------------------------------ | ------------- |
| 单一二次因式 | $\frac{P(x)}{(x-a)(x^2+1)}$ | $\frac{A}{x-a} + \frac{Bx+C}{x^2+1}$ | 二次项分子为一次式     |
| 纯二次因式  | $\frac{P(x)}{x^2+a^2}$      | $\frac{Ax+B}{x^2+a^2}$               | 分子必须设为 $Ax+B$ |

## 分母含有重复的二次因式

类似于重复的一次因式，需要把次幂展开，且每个分子都是一次多项式。

| 裂项形式 | 原式 | 裂项后表达式 | 说明 |
| ------ |------ |------ |------ |
| 平方项 | $\frac{P(x)}{(x^2+1)^2}$ | $\frac{Ax+B}{x^2+1} + \frac{Cx+D}{(x^2+1)^2}$ | 每一项分子均为一次式 |

## 数列求和中的常见裂项

在数列求和中，常利用待定系数法构造"相消"形式。

| 裂项形式 | 原式 | 裂项后表达式 | 说明 |
| ------ |------ |------ |------ |
| 一次式乘积 | $\frac{1}{n(n+1)}$ | $\frac{1}{n} - \frac{1}{n+1}$ | 基础裂项 |
| 间隔为 k | $\frac{1}{n(n+k)}$ | $\frac{1}{k} \left( \frac{1}{n} - \frac{1}{n+k} \right)$ | 提取公因数 $\frac{1}{k}$ |
| 三项乘积 | $\frac{1}{n(n+1)(n+2)}$ | $\frac{1}{2} \left( \frac{1}{n(n+1)} - \frac{1}{(n+1)(n+2)} \right)$ | 拆分为两项之差 |

## 快速确定系数的技巧（遮盖法）

对于**分母为互异一次因式**的情况，可以使用**遮盖法**快速求解。

### 遮盖法操作步骤

1. **求 A**：在原式中遮住 $(x-a)$，令 $x=a$，代入剩余部分计算
$A = \left. \frac{P(x)}{x-b} \right|_{x=a}$
2. **求 B**：在原式中遮住 $(x-b)$，令 $x=b$，代入剩余部分计算
$B = \left. \frac{P(x)}{x-a} \right|_{x=b}$

### 适用范围说明

| 技巧名称 | 适用场景 | 操作方法 | 示例 |
| ------ |------ |------ |------ |
| 遮盖法 | 分母为互异一次因式 | 遮住待求项的分母，将根代入原式剩余部分 | 求 $\frac{A}{x-a}$ 中的 $A$，遮住 $(x-a)$，令 $x=a$ |

**注意**：遮盖法仅适用于一次因式且次数为 1 的情况。对于二次因式或重复因式，仍需通分对比系数或代入特殊值求解。

## 总结

待定系数法的核心思想是：**将一个复杂的有理分式拆解为几个简单的部分分式之和**。掌握以上常见形式和技巧，能够有效解决大部分分式分解问题。

**关键要点**：

- 分母的每个线性因子对应一个常数分子项
- 重复因子需要包含所有次幂项
- 二次因子对应一次多项式分子
- 数列求和中常用裂项相消法
- 遮盖法可快速求解互异一次因式情况
