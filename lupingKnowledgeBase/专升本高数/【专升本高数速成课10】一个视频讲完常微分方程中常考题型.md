## 一阶微分方程

### 可分离变量型 *Content-[01:03]

**题型特征**：方程可化为  
$$
\frac{dy}{dx} = f(x) \cdot g(y)
$$

**解法步骤**：
1. 分离变量：  
   $$
   \frac{dy}{g(y)} = f(x) \, dx
   $$
2. 两边同时积分：  
   $$
   \int \frac{dy}{g(y)} = \int f(x) \, dx + C
   $$

**例题1**：求 $e^{-x} y' = 2$ 的通解。

- 改写为微分形式：$e^{-x} \frac{dy}{dx} = 2$
- 两边乘以 $e^x$：$\frac{dy}{dx} = 2 e^x$
- 乘 $dx$：$dy = 2 e^x dx$
- 积分：$\int dy = \int 2 e^x dx$
- 得：$y = 2 e^x + C$（$C$ 为任意常数）

**验证**：对 $y = 2 e^x + C$ 求导得 $y' = 2 e^x$，代入原方程 $e^{-x} \cdot 2 e^x = 2$ 成立。

**例题2**：求 $3y - x y' = 0$ 满足 $y(1) = \frac{1}{3}$ 的特解。

- 改写：$3y = x \frac{dy}{dx}$
- 分离变量：$\frac{dy}{y} = \frac{3}{x} dx$
- 积分：$\ln|y| = 3 \ln|x| + C$
- 两边取指数：$|y| = e^C |x|^3$
- 去掉绝对值：$y = \pm e^C \cdot |x|^3$
- 合并常数：$y = C x^3$（$C$ 为任意常数）
- 代入初值 $y(1) = \frac{1}{3}$：$\frac{1}{3} = C \cdot 1^3 \Rightarrow C = \frac{1}{3}$
- 特解：$y = \frac{1}{3} x^3$

### 齐次微分方程 *Content-[09:30]

**题型特征**：方程可化为 $\frac{dy}{dx} = \varphi\left(\frac{y}{x}\right)$ 的形式。  
常见标志：分子分母各项次数相同，除以 $x$ 的最高次幂后出现 $\frac{y}{x}$ 的函数。

**解法步骤**：
1. 令 $u = \frac{y}{x}$，则 $y = x u$，$y' = u + x \frac{du}{dx}$
2. 代入原方程，整理成关于 $u$ 的可分离变量方程
3. 解出 $u$，再代回 $u = \frac{y}{x}$

**例题**：求 $\frac{dy}{dx} = \frac{x^2 + y^2}{x y}$ 的通解。

- 右边分子分母同除以 $x^2$：  
  $$
  \frac{dy}{dx} = \frac{1 + \left(\frac{y}{x}\right)^2}{\frac{y}{x}}
  $$
- 令 $u = \frac{y}{x}$，则 $y' = u + x u'$，代入：
  $$
  u + x \frac{du}{dx} = \frac{1 + u^2}{u}
  $$
- 化简：右边 $= \frac{1}{u} + u$，两边 $u$ 抵消，得：
  $$
  x \frac{du}{dx} = \frac{1}{u}
  $$
- 分离变量：$u \, du = \frac{dx}{x}$
- 积分：$\frac{1}{2} u^2 = \ln|x| + C$
- 代回 $u = \frac{y}{x}$：$\frac{y^2}{2x^2} = \ln|x| + C$
- 整理：$y^2 = 2x^2 \ln|x| + 2C x^2$，即 $y^2 = 2x^2 \ln|x| + C x^2$（$C$ 为任意常数）

### 一阶线性微分方程 *Content-[14:50]

**标准形式**：  
$$
y' + P(x) y = Q(x)
$$

**特征**：$y'$ 和 $y$ 都是一次方，且为相加关系（非乘积）。

**通解公式**（需背诵）：  
$$
y = e^{-\int P(x) dx} \left( \int e^{\int P(x) dx} Q(x) \, dx + C \right)
$$

**记忆口诀**：  
$y = e^{负P积分} \left( \int e^{正P积分} Q \, dx + C \right)$

**例题**：解 $y' = \frac{1 + \ln x}{x} y + \frac{1}{x}$。

- 将方程整理为标准形式：  
  $$
  y' - \frac{1}{x} y = \frac{\ln x}{x}
  $$
  这里 $P(x) = -\frac{1}{x}$，$Q(x) = \frac{\ln x}{x}$。
- 套公式：
  $$
  y = e^{-\int (-\frac{1}{x}) dx} \left( \int e^{\int (-\frac{1}{x}) dx} \cdot \frac{\ln x}{x} \, dx + C \right)
  $$
- 计算积分：$\int -\frac{1}{x} dx = -\ln|x|$，故  
  $$
  e^{\int P dx} = e^{-\ln|x|} = \frac{1}{|x|}, \quad e^{-\int P dx} = |x|
  $$
- 由于公式内外均有绝对值，可直接去掉（解释：$|x|$ 与 $\frac{1}{|x|}$ 相乘后符号可被常数吸收），得：
  $$
  y = x \left( \int \frac{1}{x} \cdot \frac{\ln x}{x} \, dx + C \right) = x \left( \int \frac{\ln x}{x^2} \, dx + C \right)
  $$
- **注意**：此处原视频后续计算略有不匹配，实际应为：$y' - \frac{1}{x} y = \frac{\ln x}{x}$，正确套公式过程如下：

  先求 $\int P dx = \int -\frac{1}{x} dx = -\ln|x|$，则  
  $$
  e^{\int P dx} = e^{-\ln|x|} = \frac{1}{|x|}, \quad e^{-\int P dx} = |x|
  $$
  代入：
  $$
  y = |x| \left( \int \frac{1}{|x|} \cdot \frac{\ln x}{x} \, dx + C \right)
  $$
  因为 $\frac{1}{|x|} \cdot \frac{1}{x} = \frac{1}{x|x|}$，可整理为 $\frac{1}{x^2} \cdot \text{sign}(x)$，但通常处理时可直接假设 $x>0$ 并去掉绝对值（$C$ 可吸收符号变化）。简化后：
  $$
  y = x \left( \int \frac{\ln x}{x^2} \, dx + C \right)
  $$
  计算 $\int \frac{\ln x}{x^2} dx$：使用分部积分，令 $u=\ln x$，$dv = x^{-2}dx$，得：
  $$
  \int \frac{\ln x}{x^2} dx = -\frac{\ln x}{x} - \frac{1}{x} + C
  $$
  因此：
  $$
  y = x \left( -\frac{\ln x}{x} - \frac{1}{x} + C \right) = -\ln x - 1 + C x
  $$
  通解：$y = C x - \ln x - 1$（$C$ 为任意常数）

（注：原视频例题中 $Q(x)$ 为 $\frac{\ln x}{x}$，但讲解时误写为 $\frac{1+\ln x}{x} y + \frac{1}{x}$，应以上述修正为准。）

## 二阶常系数线性微分方程

### 齐次情形 *Content-[21:19]

**形式**：$y'' + p y' + q y = 0$（$p, q$ 为常数）

**解法**：写出特征方程 $r^2 + p r + q = 0$，解出特征根，根据根的情况写出通解：

| 特征根情况 | 通解形式 |
|-----------|----------|
| 两个不等实根 $r_1 \neq r_2$ | $y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$ |
| 两个相等实根 $r_1 = r_2 = r$ | $y = (C_1 + C_2 x) e^{r x}$ |
| 一对共轭复根 $r = \alpha \pm \beta i$ | $y = e^{\alpha x} (C_1 \sin \beta x + C_2 \cos \beta x)$ |

**例题1**：求 $y'' - 3y' + 2y = 0$ 的通解。

- 特征方程：$r^2 - 3r + 2 = 0 \Rightarrow (r-1)(r-2)=0$，$r_1=1, r_2=2$
- 通解：$y = C_1 e^{x} + C_2 e^{2x}$

**例题2**：求 $y'' - 6y' + 9y = 0$ 的通解。

- 特征方程：$r^2 - 6r + 9 = (r-3)^2 = 0$，$r_1 = r_2 = 3$
- 通解：$y = (C_1 + C_2 x) e^{3x}$

**例题3**：求 $y'' + 2y' + 10y = 0$ 的通解。

- 特征方程：$r^2 + 2r + 10 = 0$
- 配方：$(r+1)^2 + 9 = 0 \Rightarrow r+1 = \pm 3i$，$r = -1 \pm 3i$
- 其中 $\alpha = -1, \beta = 3$
- 通解：$y = e^{-x} (C_1 \sin 3x + C_2 \cos 3x)$

### 非齐次情形（类型一：多项式×指数） *Content-[28:56]

**形式**：$y'' + p y' + q y = e^{\lambda x} P_n(x)$，其中 $P_n(x)$ 为 $n$ 次多项式。

**解法**：非齐次通解 = 齐次通解 + 非齐次特解 $y^*$。

**特解设法**：
$$
y^* = x^k \cdot e^{\lambda x} \cdot Q_n(x)
$$
其中：
- $Q_n(x)$ 是与 $P_n(x)$ 同次的一般多项式（系数待定）
- $k$ 的取值：若 $\lambda$ 不是特征根，$k=0$；若是单重特征根，$k=1$；若是二重特征根，$k=2$

**例题**：求 $y'' + y' - 6y = e^x (3x+2)$ 的通解。

- 求齐次通解：特征方程 $r^2 + r - 6 = 0 \Rightarrow (r-2)(r+3)=0$，$r_1=2, r_2=-3$  
  齐次通解：$y_h = C_1 e^{2x} + C_2 e^{-3x}$

- 设特解：$\lambda = 1$，不是特征根，故 $k=0$  
  $P_n(x) = 3x+2$ 为一次多项式，设 $Q_1(x) = ax + b$  
  $$
  y^* = e^x (ax + b)
  $$

- 求导：
  $$
  y^{*\prime} = e^x (ax + b + a)
  $$
  $$
  y^{*\prime\prime} = e^x (ax + b + 2a)
  $$

- 代入原方程：
  $$
  e^x (ax+b+2a) + e^x (ax+b+a) - 6 e^x (ax+b) = e^x (3x+2)
  $$
  消去 $e^x$，整理系数：
  $$
  (a + a - 6a)x + (b+2a + b+a - 6b) = 3x + 2
  $$
  即：
  $$
  -4a x + (3a - 4b) = 3x + 2
  $$

- 比较系数得：
  $$
  \begin{cases}
  -4a = 3 \Rightarrow a = -\frac{3}{4} \\
  3a - 4b = 2 \Rightarrow 3(-\frac{3}{4}) - 4b = 2 \Rightarrow -\frac{9}{4} - 4b = 2 \Rightarrow -4b = \frac{17}{4} \Rightarrow b = -\frac{17}{16}
  \end{cases}
  $$

- 特解：$y^* = e^x \left( -\frac{3}{4}x - \frac{17}{16} \right)$

- 非齐次通解：
  $$
  y = C_1 e^{2x} + C_2 e^{-3x} + e^x \left( -\frac{3}{4}x - \frac{17}{16} \right)
  $$

### 非齐次情形（类型二：指数×三角） *Content-[40:14]

**形式**：$y'' + p y' + q y = e^{\lambda x} (A \sin \beta x + B \cos \beta x)$ 或 $e^{\lambda x} \sin \beta x$ 或 $e^{\lambda x} \cos \beta x$。

**特解设法**：
$$
y^* = x^k e^{\lambda x} (a \sin \beta x + b \cos \beta x)
$$
其中：
- $k$ 的取值：若 $\lambda \pm i\beta$ 不是特征根，则 $k=0$；若是特征根，则 $k=1$（注意：由于共轭复根成对出现，只要 $\lambda \pm i\beta$ 中有一个是特征根，即应取 $k=1$）
- $a, b$ 为待定系数

**例题**：求 $y'' + 9y = 52 e^x \sin 2x$ 的通解。

- 齐次方程：$y'' + 9y = 0$，特征方程 $r^2 + 9 = 0 \Rightarrow r = \pm 3i$  
  $\alpha = 0, \beta = 3$，齐次通解：$y_h = C_1 \sin 3x + C_2 \cos 3x$

- 自由项：$52 e^x \sin 2x$，对应 $\lambda = 1, \beta = 2$  
  判断：$\lambda \pm i\beta = 1 \pm 2i$，特征根为 $\pm 3i$，不相等，故 $k=0$

- 设特解：
  $$
  y^* = e^x (a \sin 2x + b \cos 2x)
  $$

- 求导（利用乘积求导法则，这里直接给出关键结果）：
  $$
  y^{*\prime} = e^x (a \sin 2x + b \cos 2x) + e^x (2a \cos 2x - 2b \sin 2x)
  $$
  $$
  = e^x \left[ (a - 2b) \sin 2x + (b + 2a) \cos 2x \right]
  $$
  $$
  y^{*\prime\prime} = e^x \left[ (a - 2b) \sin 2x + (b + 2a) \cos 2x \right] + e^x \left[ 2(a - 2b) \cos 2x - 2(b + 2a) \sin 2x \right]
  $$
  $$
  = e^x \left[ (a - 2b - 2b - 4a) \sin 2x + (b + 2a + 2a - 4b) \cos 2x \right]
  $$
  $$
  = e^x \left[ (-3a - 4b) \sin 2x + (4a - 3b) \cos 2x \right]
  $$

- 代入 $y'' + 9y = 52 e^x \sin 2x$：
  $$
  e^x \left[ (-3a - 4b) \sin 2x + (4a - 3b) \cos 2x \right] + 9 e^x (a \sin 2x + b \cos 2x) = 52 e^x \sin 2x
  $$
  消去 $e^x$，合并：
  $$
  [(-3a - 4b) + 9a] \sin 2x + [(4a - 3b) + 9b] \cos 2x = 52 \sin 2x
  $$
  即：
  $$
  (6a - 4b) \sin 2x + (4a + 6b) \cos 2x = 52 \sin 2x
  $$

- 比较系数：
  $$
  \begin{cases}
  6a - 4b = 52 \\
  4a + 6b = 0
  \end{cases}
  $$
  解得：
  $$
  \begin{cases}
  a = 6 \\
  b = -4
  \end{cases}
  $$

- 特解：$y^* = e^x (6 \sin 2x - 4 \cos 2x)$

- 非齐次通解：
  $$
  y = C_1 \sin 3x + C_2 \cos 3x + e^x (6 \sin 2x - 4 \cos 2x)
  $$
