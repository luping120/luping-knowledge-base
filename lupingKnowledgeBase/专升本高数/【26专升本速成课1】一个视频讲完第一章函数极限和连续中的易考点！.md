# 第一章：函数、极限与连续

## 1. 奇偶性 [原片 @ 00:15](https://www.bilibili.com/video/BV1bez4BpE8f?t=15)

**核心方法**
1.  **定义法**：前提是定义域关于原点对称。计算 $f(-x)$。
    *   若 $f(-x) = f(x)$，则为偶函数。
    *   若 $f(-x) = -f(x)$，则为奇函数。
2.  **运算法则（秒杀法）**：
    *   **记忆技巧**：将奇函数看作 $-1$，偶函数看作 $1$，运算结果的正负对应奇偶性。
    *   **加减**：奇 ± 奇 = 奇；偶 ± 偶 = 偶。
    *   **乘除**：
        *   奇 × 奇 = 偶 ( $-1 \times -1 = 1$)
        *   偶 × 奇 = 奇 ( $1 \times -1 = -1$)
        *   偶 × 偶 = 偶
    *   **复合**（“有偶则偶，全奇才奇”）：
        *   外层偶函数 $f$，复合 $f(g(x))$ 后必为偶函数。
        *   外层奇函数 $f$，复合 $f(g(x))$ 后的奇偶性与内层 $g(x)$ 相同。
        *   内层偶函数 $g$，复合 $f(g(x))$ 后必为偶函数。

**常见奇偶函数**
*   **奇函数**：$y = \sin x$, $y = x$, $y = x^3$, $y = \tan x$, $y = \arctan x$, $y = \arcsin x$。
*   **偶函数**：$y = x^2$, $y = |x|$, $y = \cos x$, $y^{2n}$（如 $x^4, x^6$）。

## 2. 定义域 [原片 @ 07:25](https://www.bilibili.com/video/BV1bez4BpE8f?t=445)

**五类天然定义域（求交集）**
1.  **分母** $\neq 0$。
2.  **偶次根式下**（如 $\sqrt{\square}, \sqrt[4]{\square}$）$\geq 0$。
3.  **对数函数**（$\ln \square, \log_a \square$）真数 $> 0$。
4.  **反正弦、反余弦** $\arcsin \square, \arccos \square$：$-1 \leq \square \leq 1$。
5.  **正切、余切** $\tan \square, \cot \square$：$\square \neq k\pi + \frac{\pi}{2}$ 等。

**解题步骤**：识别函数形式中的“框”$\square$，对“框”施加上述限制，解不等式组，取交集。

## 3. 判断函数是否相同 [原片 @ 12:21](https://www.bilibili.com/video/BV1bez4BpE8f?t=741)

**两相同原则**：定义域 **且** （化简后的）对应法则完全相同。

**例题判断逻辑**：
*   A: $f(x)=1$, $g(x)=\sin^2 x + \cos^2 x=1$，定义域均为 $\mathbb{R}$，法则相同。✔
*   B: $f(x)=x$, $g(x)=\sqrt{x^2}=|x|$，法则不同。✘
*   C: $f(x)=x$, $g(x)=\frac{x^2}{x}$，定义域不同（$g(x)$ 要求 $x \neq 0$）。✘
*   D: $f(x)=\ln x (x>0)$, $g(x)=2\ln x (x \neq 0)$，定义域不同。✘

## 4. 求函数的表达式 [原片 @ 16:10](https://www.bilibili.com/video/BV1bez4BpE8f?t=970)

**类型一：已知 $f(x)$，求 $f(\text{狗})$**
*   **方法**：直接代入。将 $f(x)$ 表达式中的所有 $x$ 替换为“狗”。
*   **例**：$f(x)=1+3e^x$，则 $f(x^2)=1+3e^{x^2}$。

**类型二：已知 $f(\text{狗})$，求 $f(x)$**
*   **方法**：换元法。
    1.  令 $\text{狗} = t$。
    2.  反解出 $x = \varphi(t)$。
    3.  代入原式 $f(\text{狗})=f(t)=...$，得到关于 $t$ 的表达式。
    4.  将 $t$ 换回 $x$，得 $f(x)$。
*   **例**：$f(x-2019)=x^2 - 2019x + 2020$，求 $f(x)$。
    解：令 $t = x - 2019$，则 $x = t + 2019$。代入：
    $f(t) = (t+2019)^2 - 2019(t+2019) + 2020 = t^2 + 2019t + 2020$。
    故 $f(x) = x^2 + 2019x + 2020$。

# 第二章：极限的计算

**核心思想**：先判断未定式类型，再选用对应方法。

## 1. $0/0$ 型 [原片 @ 20:41](https://www.bilibili.com/video/BV1bez4BpE8f?t=1241)

**解题步骤**：先尝试等价无穷小替换，再使用洛必达法则 (L‘Hospital‘s Rule)。

**常用等价无穷小（$x \to 0$ 时）**：
*   $\sin x \sim x$
*   $\arcsin x \sim x$
*   $\tan x \sim x$
*   $\arctan x \sim x$
*   $e^x - 1 \sim x$
*   $\ln(1+x) \sim x$
*   $1 - \cos x \sim \frac{1}{2}x^2$
*   $(1+x)^\alpha - 1 \sim \alpha x$

**推广**：当 $\square \to 0$ 时，$\sin \square \sim \square$，$1-\cos \square \sim \frac{1}{2}\square^2$ 等同样成立。

**⚠️ 注意**：等价无穷小**只能用于乘除因子**，不能用于加减项。

**洛必达法则**：若 $\lim \frac{f(x)}{g(x)}$ 为 $0/0$ 或 $\infty/\infty$ 型，则 $\lim \frac{f(x)}{g(x)} = \lim \frac{f'(x)}{g'(x)}$（前提后者极限存在或为无穷）。可连续使用。

## 2. $\infty/\infty$ 型 [原片 @ 33:56](https://www.bilibili.com/video/BV1bez4BpE8f?t=2036)

**方法一：洛必达法则**（同上）。

**方法二：抓大头法（等价无穷大）**
*   **思想**：当 $x \to \infty$ 时，多项式或复合函数的值由增长最快的项（“大哥”）决定。
*   **常见函数趋向无穷的速度比较**：$\ln x \ll x^n \ll a^x (a>1)$。（对数函数 < 幂函数 < 指数函数）
*   **操作**：分子分母分别只保留趋向无穷最快的项。
    *   **例**：$\lim_{x\to\infty} \frac{2x^2+3x+5}{3x^2+7} = \lim_{x\to\infty} \frac{2x^2}{3x^2} = \frac{2}{3}$。
    *   **例**：$\lim_{x\to\infty} \frac{2x^2+3x+5}{3x^2 + e^x} = \lim_{x\to\infty} \frac{2x^2}{e^x} = 0$（因为指数函数比幂函数快）。

## 3. $0 \cdot \infty$ 型 [原片 @ 39:51](https://www.bilibili.com/video/BV1bez4BpE8f?t=2391)

**方法：下放转化**
*   将乘积中的一项下放到分母，转化为 $0/0$ 或 $\infty/\infty$ 型。
    *   $0 \cdot \infty = \frac{0}{(1/\infty)} = \frac{0}{0}$
    *   或 $0 \cdot \infty = \frac{\infty}{(1/0)} = \frac{\infty}{\infty}$
*   **原则**：谁复杂（求导麻烦）谁下放。
*   **例**：$\lim_{x \to 0^+} x \ln x = \lim_{x \to 0^+} \frac{\ln x}{1/x} \xRightarrow{\text{洛或等价}} 0$。

## 4. $\infty - \infty$ 型 [原片 @ 41:05](https://www.bilibili.com/video/BV1bez4BpE8f?t=2465)

**方法一：通分**（适用于分式相减）。
**方法二：有理化**（适用于含根式相减，$a-b$ 型分子分母同乘 $a+b$）。

**例（通分）**：$\lim_{x \to 0} (\frac{1}{x} - \frac{1}{\sin x}) = \lim_{x \to 0} \frac{\sin x - x}{x \sin x} \xRightarrow{0/0型} ...$
**例（有理化）**：$\lim_{x \to +\infty} (\sqrt{x^2+x} - x) = \lim_{x \to +\infty} \frac{(\sqrt{x^2+x} - x)(\sqrt{x^2+x} + x)}{\sqrt{x^2+x} + x} = \lim_{x \to +\infty} \frac{x}{\sqrt{x^2+x} + x} \xRightarrow{\text{抓大头}} \frac{1}{2}$

## 5. $1^\infty$ 型 [原片 @ 46:01](https://www.bilibili.com/video/BV1bez4BpE8f?t=2761)

**方法：万能公式（重点记忆）**
若 $\lim f(x) = 1$， $\lim g(x) = \infty$，则
$$
\lim [f(x)]^{g(x)} = e^{\lim [g(x) \cdot (f(x)-1)]}
$$
**口诀**：等于 $e$ 的（指数 $g(x)$ 乘 底数减1）次方。

**例**：
1.  $\lim_{x \to \infty} (1+\frac{1}{x})^{2x} = e^{\lim_{x \to \infty} [2x \cdot (1+\frac{1}{x}-1)]} = e^{\lim_{x \to \infty} 2} = e^2$。
2.  $\lim_{x \to 0} (1+2x)^{\frac{1}{x}} = e^{\lim_{x \to 0} [\frac{1}{x} \cdot (1+2x-1)]} = e^{\lim_{x \to 0} 2} = e^2$。

## 6. $0^0$ 型与 $\infty^0$ 型 [原片 @ 49:23](https://www.bilibili.com/video/BV1bez4BpE8f?t=2963)

**方法：取对数法（幂指函数通用）**
对于 $\lim u(x)^{v(x)}$，利用公式 $u^v = e^{v \ln u}$。
$$
\lim u(x)^{v(x)} = e^{\lim [v(x) \ln u(x)]}
$$
此时指数部分化为 $0 \cdot \infty$ 型，再用前述方法解决。

**例（$0^0$型）**：$\lim_{x \to 0^+} x^x = e^{\lim_{x \to 0^+} (x \ln x)} = e^0 = 1$。

## 7. 数列极限 [原片 @ 53:36](https://www.bilibili.com/video/BV1bez4BpE8f?t=3216)

**方法：放缩法 + 夹逼准则**
1.  **放缩**：将数列 $a_n$ 的每一项放大或缩小。
    *   **缩小**：将所有项都换成它们中最小的项。
    *   **放大**：将所有项都换成它们中最大的项。
    得到：缩小和 $\le a_n \le$ 放大和。
2.  **夹逼**：计算缩小和与放大和的极限。如果它们极限相等（设为 $A$），则原数列极限 $\lim a_n = A$。

**例**：求 $\lim_{n \to \infty} (\frac{1}{\sqrt{9n^2+1}} + \frac{1}{\sqrt{9n^2+2}} + ... + \frac{1}{\sqrt{9n^2+n}})$。
解：最小项为 $\frac{1}{\sqrt{9n^2+n}}$，最大项为 $\frac{1}{\sqrt{9n^2+1}}$。
故 $n \cdot \frac{1}{\sqrt{9n^2+n}} \le a_n \le n \cdot \frac{1}{\sqrt{9n^2+1}}$。
计算左右极限（抓大头）：$\lim_{n \to \infty} \frac{n}{3n} = \frac{1}{3}$， $\lim_{n \to \infty} \frac{n}{3n} = \frac{1}{3}$。
由夹逼准则得 $\lim a_n = \frac{1}{3}$。

# 第三章：连续性与间断点

## 1. 连续性 [原片 @ 59:30](https://www.bilibili.com/video/BV1bez4BpE8f?t=3570)

**定义**：$\lim_{x \to x_0} f(x) = f(x_0)$。即函数在该点的极限值等于函数值。
*   **左连续**：$\lim_{x \to x_0^-} f(x) = f(x_0)$。
*   **右连续**：$\lim_{x \to x_0^+} f(x) = f(x_0)$。
*   函数在 $x_0$ 连续 $\iff$ 在 $x_0$ 既左连续又右连续。

**关系**：连续 $\Rightarrow$ 极限存在；极限存在 $\nRightarrow$ 连续（还需等于函数值）。

## 2. 间断点及其分类 *Content-[01:09:48]

**第一类间断点（左右极限都存在）**
*   **可去间断点**：左右极限相等，但不等于函数值（或函数在该点无定义）。`lim = A ≠ f(x0)`
*   **跳跃间断点**：左右极限存在但不相等。`lim⁻ ≠ lim⁺`

**第二类间断点（左右极限至少一个不存在）**
*   **无穷间断点**：左右极限至少有一个为无穷大（如 $\lim = \infty$）。
*   **振荡间断点**：极限振荡不存在（如 $\lim_{x \to 0} \sin \frac{1}{x}$）。

**寻找间断点**：通常关注**分段函数的分段点**和**函数无定义的点**（如分母为零的点）。

**例**：$f(x) = \frac{x-3}{x^2-9}$，无定义点为 $x=3$ 和 $x=-3$。
*   $x=3$: $\lim_{x \to 3} f(x) = \lim_{x \to 3} \frac{1}{x+3} = \frac{1}{6}$（极限存在），为**可去间断点**。
*   $x=-3$: $\lim_{x \to -3} f(x) = \infty$，为**无穷间断点**。
