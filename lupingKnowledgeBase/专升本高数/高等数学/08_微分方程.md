## 8.1 微分方程的基本概念

### 📖 基本概念

**微分方程**：含有未知函数导数或微分的方程。
**阶**：方程中出现的未知函数最高阶导数的阶数。
**解**：代入方程使其成为恒等式的函数。
**通解**：含有与方程阶数相同个数的独立任意常数的解。
**特解**：不含任意常数或常数已确定的解。
**初始条件**：确定特解的条件。

**线性与非线性**：
- **线性微分方程**：未知函数及其各阶导数都是一次的
- **非线性微分方程**：不满足线性条件

### 📊 微分方程分类

| 分类依据 | 类型 | 示例 |
|----------|------|------|
| **阶数** | 一阶、二阶、高阶 | $y' = f(x,y)$（一阶）<br>$y'' + p(x)y' + q(x)y = f(x)$（二阶） |
| **线性性** | 线性、非线性 | $y'' + y = 0$（线性）<br>$y' + y^2 = 0$（非线性） |
| **系数性质** | 常系数、变系数 | $y'' + 2y' + y = 0$（常系数）<br>$y'' + xy' + y = 0$（变系数） |

### 💡 验证解的方法
1. 将函数及其各阶导数代入原方程
2. 检查是否成为恒等式
3. 检查是否满足初始条件（特解）

### 🧮 示例演示

**例1**：验证 $y = C_1 e^x + C_2 e^{-x}$ 是方程 $y'' - y = 0$ 的通解
- $y' = C_1 e^x - C_2 e^{-x}$，$y'' = C_1 e^x + C_2 e^{-x}$
- 代入：$y'' - y = (C_1 e^x + C_2 e^{-x}) - (C_1 e^x + C_2 e^{-x}) = 0$
- 含有两个独立任意常数，是二阶方程的通解

**例2**：求微分方程 $y' = 2x$ 满足初始条件 $y(0)=1$ 的特解
- 通解：$y = \int 2x dx = x^2 + C$
- 代入 $y(0)=1$：$1 = 0 + C$ ⇒ $C=1$
- 特解：$y = x^2 + 1$

## 8.2 一阶微分方程

### 📚 可分离变量方程

**形式**：$\frac{dy}{dx} = f(x)g(y)$ 或 $M(x)N(y)dx + P(x)Q(y)dy = 0$

**解法**：分离变量，两边积分
$$ \frac{dy}{g(y)} = f(x) dx \quad \Rightarrow \quad \int \frac{dy}{g(y)} = \int f(x) dx + C $$

**记忆口诀**："变量分离左右分，两边积分加常数"

### 📚 齐次方程

**形式**：$\frac{dy}{dx} = f\left(\frac{y}{x}\right)$

**解法**：令 $u = \frac{y}{x}$，则 $y = ux$，$y' = u + xu'$
代入得：$u + x\frac{du}{dx} = f(u)$ ⇒ 分离变量求解

**记忆口诀**："y比x的函数式，令u等于y比x"

### 📚 一阶线性微分方程

**标准形式**：$\frac{dy}{dx} + P(x)y = Q(x)$

**解法**：
1. **齐次方程**：$\frac{dy}{dx} + P(x)y = 0$ ⇒ 分离变量得 $y = Ce^{-\int P(x) dx}$
2. **非齐次方程**：常数变易法或公式法
   $$ y = e^{-\int P(x) dx} \left[ \int Q(x) e^{\int P(x) dx} dx + C \right] $$

**记忆口诀**："一阶线性标准形，公式套用要记清"

### 📚 伯努利方程

**形式**：$\frac{dy}{dx} + P(x)y = Q(x)y^n$ （$n \neq 0,1$）

**解法**：令 $z = y^{1-n}$，化为线性方程

**记忆口诀**："y的n次伯努利，令z等于y的1-n次"

### 🧮 示例演示

**例1（可分离变量）**：解 $\frac{dy}{dx} = \frac{x}{y}$
- 分离变量：$y dy = x dx$
- 积分：$\frac{1}{2}y^2 = \frac{1}{2}x^2 + C$ ⇒ $y^2 = x^2 + C_1$

**例2（齐次方程）**：解 $\frac{dy}{dx} = \frac{y}{x} + \tan\frac{y}{x}$
- 令 $u = \frac{y}{x}$，则 $y = ux$，$y' = u + xu'$
- $u + xu' = u + \tan u$ ⇒ $x\frac{du}{dx} = \tan u$
- 分离变量：$\frac{du}{\tan u} = \frac{dx}{x}$ ⇒ $\ln|\sin u| = \ln|x| + C$
- $\sin u = Cx$ ⇒ $\sin\frac{y}{x} = Cx$

**例3（一阶线性）**：解 $y' + \frac{2}{x}y = \frac{\sin x}{x^2}$
- $P(x) = \frac{2}{x}$，$Q(x) = \frac{\sin x}{x^2}$
- $\int P(x) dx = 2\ln|x| = \ln x^2$
- $e^{\int P(x) dx} = x^2$，$e^{-\int P(x) dx} = \frac{1}{x^2}$
- $y = \frac{1}{x^2} \left[ \int \frac{\sin x}{x^2} \cdot x^2 dx + C \right] = \frac{1}{x^2} (-\cos x + C)$

**例4（伯努利方程）**：解 $y' + y = xy^2$
- $n=2$，令 $z = y^{1-2} = y^{-1}$，则 $y = z^{-1}$，$y' = -z^{-2}z'$
- 代入：$-z^{-2}z' + z^{-1} = x z^{-2}$ ⇒ $z' - z = -x$（线性方程）
- 解得：$z = Ce^x + x + 1$ ⇒ $y = \frac{1}{Ce^x + x + 1}$

## 8.3 可降阶的高阶微分方程

### 📚 $y^{(n)} = f(x)$ 型

**形式**：$y^{(n)} = f(x)$

**解法**：连续积分 $n$ 次
$$ y^{(n-1)} = \int f(x) dx + C_1 $$
$$ y^{(n-2)} = \int \left[ \int f(x) dx + C_1 \right] dx + C_2 $$
依此类推。

### 📚 $y'' = f(x, y')$ 型（不显含 $y$）

**形式**：$y'' = f(x, y')$

**解法**：令 $p = y'$，则 $y'' = p'$，方程化为
$$ p' = f(x, p) $$
这是一阶方程，解出 $p = \varphi(x, C_1)$，再积分得 $y$。

### 📚 $y'' = f(y, y')$ 型（不显含 $x$）

**形式**：$y'' = f(y, y')$

**解法**：令 $p = y'$，则 $y'' = \frac{dp}{dx} = \frac{dp}{dy} \cdot \frac{dy}{dx} = p \frac{dp}{dy}$
方程化为：
$$ p \frac{dp}{dy} = f(y, p) $$
这是关于 $y, p$ 的一阶方程。

### 💡 降阶法总结

| 方程类型 | 降阶方法 | 新方程 |
|----------|----------|--------|
| $y^{(n)} = f(x)$ | 直接积分 $n$ 次 | - |
| $y'' = f(x, y')$ | 令 $p = y'$ | $p' = f(x, p)$ |
| $y'' = f(y, y')$ | 令 $p = y'$，$y'' = p\frac{dp}{dy}$ | $p\frac{dp}{dy} = f(y, p)$ |

### 🧮 示例演示

**例1**：解 $y''' = e^{2x} + \cos x$
- 积分：$y'' = \frac{1}{2}e^{2x} + \sin x + C_1$
- 再积分：$y' = \frac{1}{4}e^{2x} - \cos x + C_1 x + C_2$
- 再积分：$y = \frac{1}{8}e^{2x} - \sin x + \frac{1}{2}C_1 x^2 + C_2 x + C_3$

**例2**：解 $y'' = \frac{y'}{x} + x$（不显含 $y$）
- 令 $p = y'$，则 $p' = \frac{p}{x} + x$
- 一阶线性方程：$p' - \frac{1}{x}p = x$
- 解得：$p = x\left(\int 1 dx + C_1\right) = x(x + C_1) = x^2 + C_1 x$
- $y = \int (x^2 + C_1 x) dx = \frac{1}{3}x^3 + \frac{1}{2}C_1 x^2 + C_2$

**例3**：解 $y'' = 2y y'$（不显含 $x$）
- 令 $p = y'$，则 $y'' = p\frac{dp}{dy}$
- $p\frac{dp}{dy} = 2y p$ ⇒ 若 $p \neq 0$，$\frac{dp}{dy} = 2y$
- $p = y^2 + C_1$ ⇒ $\frac{dy}{dx} = y^2 + C_1$
- 分离变量：$\frac{dy}{y^2 + C_1} = dx$，积分得解

## 8.4 二阶常系数线性微分方程

### 📖 基本概念

**标准形式**：
$$ y'' + py' + qy = f(x) $$
其中 $p, q$ 为常数。

- **齐次方程**：$f(x) \equiv 0$，即 $y'' + py' + qy = 0$
- **非齐次方程**：$f(x) \not\equiv 0$

### 📚 齐次方程解法

**特征方程**：$r^2 + pr + q = 0$

**通解形式**：

| 特征根情况 | 通解形式 |
|------------|----------|
| **两个不等实根** $r_1 \neq r_2$ | $y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$ |
| **相等实根** $r_1 = r_2 = r$ | $y = (C_1 + C_2 x) e^{rx}$ |
| **共轭复根** $r = \alpha \pm i\beta$ | $y = e^{\alpha x} (C_1 \cos \beta x + C_2 \sin \beta x)$ |

**记忆口诀**：
- "不等实根指数和"
- "相等实根乘x"
- "共轭复根三角乘指数"

### 📚 非齐次方程解法

**解的结构**：$y = y^* + \bar{y}$
- $y^*$：非齐次方程的一个特解
- $\bar{y}$：对应齐次方程的通解

**特解求法**（待定系数法）：

| $f(x)$ 形式 | 特解形式（$\lambda$ 不是特征根） | 特解形式（$\lambda$ 是 $k$ 重特征根） |
|-------------|----------------------------------|--------------------------------------|
| $P_n(x)$（$n$ 次多项式） | $Q_n(x)$（$n$ 次多项式） | $x^k Q_n(x)$ |
| $P_n(x) e^{\lambda x}$ | $Q_n(x) e^{\lambda x}$ | $x^k Q_n(x) e^{\lambda x}$ |
| $e^{\alpha x} (P_m(x) \cos \beta x + P_n(x) \sin \beta x)$ | $e^{\alpha x} (Q_l(x) \cos \beta x + R_l(x) \sin \beta x)$<br>其中 $l = \max(m,n)$ | $x^k e^{\alpha x} (Q_l(x) \cos \beta x + R_l(x) \sin \beta x)$ |

### 💡 求解步骤

1. 写出特征方程，求特征根
2. 写出齐次方程通解 $\bar{y}$
3. 根据 $f(x)$ 形式设特解 $y^*$
4. 将 $y^*$ 代入原方程确定系数
5. 通解 $y = \bar{y} + y^*$
6. 如有初始条件，确定常数

### 🧮 示例演示

**例1（齐次）**：解 $y'' - 5y' + 6y = 0$
- 特征方程：$r^2 - 5r + 6 = 0$ ⇒ $r_1 = 2$，$r_2 = 3$
- 通解：$y = C_1 e^{2x} + C_2 e^{3x}$

**例2（齐次，复根）**：解 $y'' + 4y' + 13y = 0$
- 特征方程：$r^2 + 4r + 13 = 0$ ⇒ $r = -2 \pm 3i$
- 通解：$y = e^{-2x} (C_1 \cos 3x + C_2 \sin 3x)$

**例3（非齐次）**：解 $y'' - 2y' + y = xe^x$
- 齐次方程：$r^2 - 2r + 1 = 0$ ⇒ $r = 1$（二重根）
- 齐次通解：$\bar{y} = (C_1 + C_2 x)e^x$
- $f(x) = xe^x$，$\lambda=1$ 是二重特征根，$k=2$
- 设特解：$y^* = x^2 (Ax+B)e^x = (Ax^3+Bx^2)e^x$
- 求导代入确定系数：得 $A=\frac{1}{6}$，$B=0$
- 通解：$y = (C_1 + C_2 x + \frac{1}{6}x^3)e^x$

**例4（非齐次，三角函数）**：解 $y'' + y = \sin x$
- 齐次方程：$r^2 + 1 = 0$ ⇒ $r = \pm i$
- 齐次通解：$\bar{y} = C_1 \cos x + C_2 \sin x$
- $f(x) = \sin x$，$\alpha=0$，$\beta=1$，$\alpha \pm i\beta = \pm i$ 是特征根
- 设特解：$y^* = x(A\cos x + B\sin x)$
- 求导代入确定系数：得 $A=-\frac{1}{2}$，$B=0$
- 通解：$y = C_1 \cos x + C_2 \sin x - \frac{1}{2}x\cos x$

## 🔑 第八章答案与解析

### 基础题解析
1. **$y^2 = \frac{2}{3}x^3 + C$ 或 $y = \pm\sqrt{\frac{2}{3}x^3 + C}$**
   - **解析**：$y dy = x^2 dx$ ⇒ $\frac{1}{2}y^2 = \frac{1}{3}x^3 + C$ ⇒ $y^2 = \frac{2}{3}x^3 + C_1$

2. **$y = \frac{-\cos x + C}{x}$**
   - **解析**：$P(x)=\frac{1}{x}$，$Q(x)=\frac{\sin x}{x}$
     $\int P(x) dx = \ln|x|$，$e^{\int P(x) dx} = x$
     $y = \frac{1}{x}\left[\int \frac{\sin x}{x} \cdot x dx + C\right] = \frac{1}{x}(-\cos x + C)$

3. **$\ln\left|\frac{y}{x}\right| = \ln|x| + C$ 或 $y = \pm x\sqrt{C_1 x}$**
   - **解析**：令 $u = \frac{y}{x}$，则 $y = ux$，$y' = u + xu'$
     $u + xu' = u + \frac{1}{u}$ ⇒ $xu' = \frac{1}{u}$ ⇒ $u du = \frac{dx}{x}$
     $\frac{1}{2}u^2 = \ln|x| + C$ ⇒ $u^2 = 2\ln|x| + C_1$ ⇒ $\frac{y^2}{x^2} = 2\ln|x| + C_1$

4. **$y = \frac{1}{3}x^3 + 1$**
   - **解析**：$y'' = 2x$ ⇒ $y' = x^2 + C_1$
     由 $y'(0)=0$ ⇒ $C_1=0$ ⇒ $y' = x^2$
     $y = \frac{1}{3}x^3 + C_2$，由 $y(0)=1$ ⇒ $C_2=1$

5. **$y = (C_1 + C_2 x)e^{2x}$**
   - **解析**：特征方程 $r^2 - 4r + 4 = 0$ ⇒ $(r-2)^2=0$ ⇒ $r=2$（二重根）
     通解：$y = (C_1 + C_2 x)e^{2x}$

### 提高题解析
6. **$y = \frac{1}{Ce^x - e^x}$**
   - **解析**：令 $z = y^{-1}$，则 $y = z^{-1}$，$y' = -z^{-2}z'$
     代入：$-z^{-2}z' + z^{-1} = z^{-2}e^x$ ⇒ $z' - z = -e^x$
     线性方程：$z = e^x(C - x)$
     $y = \frac{1}{e^x(C - x)} = \frac{1}{Ce^x - xe^x}$

7. **$y = C_1 \cos x + C_2 \sin x + x^2 - 2$**
   - **解析**：齐次方程 $y''+y=0$ 通解：$\bar{y} = C_1 \cos x + C_2 \sin x$
     设特解 $y^* = Ax^2+Bx+C$，代入得：$2A + Ax^2+Bx+C = x^2$
     比较系数：$A=1$，$B=0$，$2A+C=0$ ⇒ $C=-2$
     特解：$y^* = x^2 - 2$，通解：$y = C_1 \cos x + C_2 \sin x + x^2 - 2$

8. **$y = 2\cos 2x$**
   - **解析**：特征方程 $r^2 + 4 = 0$ ⇒ $r = \pm 2i$
     通解：$y = C_1 \cos 2x + C_2 \sin 2x$
     由 $y(0)=2$ ⇒ $C_1 = 2$
     $y' = -2C_1 \sin 2x + 2C_2 \cos 2x$，$y'(0)=0$ ⇒ $2C_2 = 0$ ⇒ $C_2=0$
     特解：$y = 2\cos 2x$
