## 4.1 不定积分的概念与性质

### 📖 不定积分概念

**原函数定义**：
> 设 $f(x)$ 在区间 $I$ 上有定义，如果存在函数 $F(x)$，使对任意 $x \in I$ 都有：
> $$ F'(x) = f(x) $$
> 则称 $F(x)$ 为 $f(x)$ 在区间 $I$ 上的一个**原函数**。

**不定积分定义**：
> 函数 $f(x)$ 的所有原函数的集合称为 $f(x)$ 的**不定积分**，记作：
> $$ \int f(x) dx = F(x) + C $$
> 其中 $C$ 为任意常数。

**记忆**：积分是导数的逆运算。

### 🔍 基本性质

1. **线性性质**：
   $$ \int [k_1 f(x) + k_2 g(x)] dx = k_1 \int f(x) dx + k_2 \int g(x) dx $$

2. **求导与积分互逆**：
   $$ \frac{d}{dx} \left[ \int f(x) dx \right] = f(x) $$
   $$ \int F'(x) dx = F(x) + C $$

### 📚 基本积分公式（必须牢记！）

| 函数类型 | 积分公式 | 记忆口诀 |
|----------|----------|----------|
| **幂函数** | $\int x^\alpha dx = \frac{x^{\alpha+1}}{\alpha+1} + C$ ($\alpha \neq -1$) | "幂加1，除以新幂" |
| **指数函数** | $\int a^x dx = \frac{a^x}{\ln a} + C$<br>$\int e^x dx = e^x + C$ | "指数照抄，除以ln底"<br>"e的x次积自身" |
| **三角函数** | $\int \sin x dx = -\cos x + C$<br>$\int \cos x dx = \sin x + C$<br>$\int \sec^2 x dx = \tan x + C$<br>$\int \csc^2 x dx = -\cot x + C$ | "正弦积负余弦，余弦积正弦"<br>"正割方积正切，余割方积负余切" |
| **分式函数** | $\int \frac{1}{x} dx = \ln|x| + C$<br>$\int \frac{1}{1+x^2} dx = \arctan x + C$<br>$\int \frac{1}{\sqrt{1-x^2}} dx = \arcsin x + C$ | "1/x积ln|x|"<br>"1加x方分之1积反正切"<br>"根号1减x方分之1积反正弦" |

### 💡 积分技巧
1. **先化简后积分**：利用代数、三角恒等式化简被积函数
2. **拆项积分**：将被积函数拆分为多个简单函数的和
3. **凑微分法**：观察被积函数中是否有导数关系

### 🧮 示例演示
**例1**：求 $\int (3x^2 + 2x - 1) dx$
- $\int 3x^2 dx = x^3$，$\int 2x dx = x^2$，$\int (-1) dx = -x$
- 结果：$x^3 + x^2 - x + C$

**例2**：求 $\int \frac{x^2 + 1}{x} dx$
- 化简：$\frac{x^2 + 1}{x} = x + \frac{1}{x}$
- $\int x dx = \frac{1}{2}x^2$，$\int \frac{1}{x} dx = \ln|x|$
- 结果：$\frac{1}{2}x^2 + \ln|x| + C$

## 4.2 换元积分法

### 🔄 第一类换元法（凑微分法）

**方法**：设 $u = \varphi(x)$ 可导，则
$$ \int f[\varphi(x)] \varphi'(x) dx = \int f(u) du $$

**步骤**：
1. 识别出 $\varphi'(x) dx$ 的形式
2. 令 $u = \varphi(x)$
3. 换元积分
4. 回代 $u = \varphi(x)$

**记忆口诀**："看到复合求导逆，凑出微分换元积"

### 🔄 第二类换元法（变量代换法）

**方法**：设 $x = \psi(t)$ 单调可导，$\psi'(t) \neq 0$，则
$$ \int f(x) dx = \int f[\psi(t)] \psi'(t) dt $$

**常用代换**：
1. **三角代换**：
   - $\sqrt{a^2 - x^2}$：令 $x = a\sin t$ 或 $x = a\cos t$
   - $\sqrt{a^2 + x^2}$：令 $x = a\tan t$
   - $\sqrt{x^2 - a^2}$：令 $x = a\sec t$
2. **倒代换**：$x = \frac{1}{t}$，用于分母次数较高时
3. **指数代换**：$e^x = t$，用于含 $e^x$ 的积分

### 💡 凑微分常用公式

| 形式 | 凑微分 |
|------|--------|
| $f(ax+b)$ | $\frac{1}{a} d(ax+b)$ |
| $f(e^x)$ | $d(e^x)$ |
| $f(\ln x)$ | $\frac{1}{x} d(\ln x)$ |
| $f(\sin x)$ | $\cos x d(\sin x)$ |
| $f(\tan x)$ | $\sec^2 x d(\tan x)$ |

### 🧮 示例演示
**例1（第一类换元）**：求 $\int \cos(2x+1) dx$
- 令 $u = 2x+1$，则 $du = 2dx$ ⇒ $dx = \frac{1}{2}du$
- $\int \cos u \cdot \frac{1}{2} du = \frac{1}{2} \sin u + C = \frac{1}{2} \sin(2x+1) + C$

**例2（第二类换元）**：求 $\int \frac{dx}{\sqrt{x^2+4}}$
- 令 $x = 2\tan t$，则 $dx = 2\sec^2 t dt$，$\sqrt{x^2+4} = 2\sec t$
- $\int \frac{2\sec^2 t dt}{2\sec t} = \int \sec t dt = \ln|\sec t + \tan t| + C$
- $\tan t = \frac{x}{2}$，$\sec t = \frac{\sqrt{x^2+4}}{2}$
- 结果：$\ln\left|\frac{x}{2} + \frac{\sqrt{x^2+4}}{2}\right| + C = \ln\left|x + \sqrt{x^2+4}\right| + C_1$

## 4.3 分部积分法

### 📝 分部积分公式

**公式**：
$$ \int u dv = uv - \int v du $$
或
$$ \int u(x) v'(x) dx = u(x) v(x) - \int u'(x) v(x) dx $$

**记忆口诀**："前导后不导的积分等于前乘后减前不导后导的积分"

### 🎯 $u$ 的选择原则（LIATE法则）

按优先级选择 $u$：
1. **L**：对数函数（$\ln x$, $\log_a x$）
2. **I**：反三角函数（$\arcsin x$, $\arctan x$ 等）
3. **A**：代数函数（$x^n$, 多项式）
4. **T**：三角函数（$\sin x$, $\cos x$ 等）
5. **E**：指数函数（$e^x$, $a^x$）

**记忆口诀**："反对幂三指"

### 🔄 循环积分

有些积分经过两次分部积分后出现原积分，形成方程求解：
$$ \int e^{ax} \sin bx dx, \quad \int e^{ax} \cos bx dx $$

**技巧**：两次分部积分后移项解方程。

### 🧮 示例演示
**例1**：求 $\int x e^x dx$
- 令 $u = x$，$dv = e^x dx$ ⇒ $du = dx$，$v = e^x$
- $\int x e^x dx = x e^x - \int e^x dx = x e^x - e^x + C = (x-1)e^x + C$

**例2**：求 $\int \ln x dx$
- 令 $u = \ln x$，$dv = dx$ ⇒ $du = \frac{1}{x} dx$，$v = x$
- $\int \ln x dx = x \ln x - \int x \cdot \frac{1}{x} dx = x \ln x - x + C$

**例3（循环积分）**：求 $\int e^x \sin x dx$
- 令 $u = \sin x$，$dv = e^x dx$ ⇒ $du = \cos x dx$，$v = e^x$
- $\int e^x \sin x dx = e^x \sin x - \int e^x \cos x dx$
- 对 $\int e^x \cos x dx$ 再次分部积分：令 $u = \cos x$，$dv = e^x dx$
- $\int e^x \cos x dx = e^x \cos x + \int e^x \sin x dx$
- 代入：$\int e^x \sin x dx = e^x \sin x - [e^x \cos x + \int e^x \sin x dx]$
- 移项：$2\int e^x \sin x dx = e^x (\sin x - \cos x)$
- 结果：$\int e^x \sin x dx = \frac{1}{2} e^x (\sin x - \cos x) + C$

## 4.4 有理函数的积分

### 📖 有理函数分解

**真分式**：分子次数 < 分母次数
**假分式**：分子次数 ≥ 分母次数

**步骤**：
1. 如果是假分式，用多项式除法化为多项式 + 真分式
2. 对真分式分母进行因式分解
3. 将真分式分解为部分分式之和

### 📚 部分分式分解形式

| 分母因式 | 部分分式形式 |
|----------|--------------|
| $(x-a)$ | $\frac{A}{x-a}$ |
| $(x-a)^k$ | $\frac{A_1}{x-a} + \frac{A_2}{(x-a)^2} + \cdots + \frac{A_k}{(x-a)^k}$ |
| $(x^2+px+q)$（不可约） | $\frac{Ax+B}{x^2+px+q}$ |
| $(x^2+px+q)^k$ | $\frac{A_1x+B_1}{x^2+px+q} + \frac{A_2x+B_2}{(x^2+px+q)^2} + \cdots + \frac{A_kx+B_k}{(x^2+px+q)^k}$ |

### 💡 系数确定方法
1. **比较系数法**：通分后比较分子系数
2. **赋值法**：代入特殊 $x$ 值（如分母的根）
3. **极限法**：取 $x \to \infty$ 等

### 🧮 示例演示
**例**：求 $\int \frac{x+1}{x^2-5x+6} dx$
1. 分母分解：$x^2-5x+6 = (x-2)(x-3)$
2. 设 $\frac{x+1}{(x-2)(x-3)} = \frac{A}{x-2} + \frac{B}{x-3}$
3. 通分：$x+1 = A(x-3) + B(x-2)$
4. 确定系数：
   - 令 $x=2$：$3 = A(-1)$ ⇒ $A = -3$
   - 令 $x=3$：$4 = B(1)$ ⇒ $B = 4$
5. 积分：
   $\int \frac{x+1}{x^2-5x+6} dx = \int \left( \frac{-3}{x-2} + \frac{4}{x-3} \right) dx = -3\ln|x-2| + 4\ln|x-3| + C$

## 4.5 三角函数有理式的积分

### 🔄 万能代换法

**万能代换**：令 $t = \tan\frac{x}{2}$，则
$$ \sin x = \frac{2t}{1+t^2}, \quad \cos x = \frac{1-t^2}{1+t^2}, \quad dx = \frac{2}{1+t^2} dt $$

**优点**：将三角有理式化为有理函数积分。
**缺点**：计算较复杂，通常作为最后手段。

### 💡 特殊情况的简化方法

1. **$R(\sin x, \cos x)$ 为奇函数**：
   - 关于 $\sin x$ 奇：令 $t = \cos x$
   - 关于 $\cos x$ 奇：令 $t = \sin x$

2. **$R(\sin^2 x, \cos^2 x)$ 型**：令 $t = \tan x$

3. **$\sin^m x \cos^n x$ 型**：
   - $m$ 为奇数：令 $t = \cos x$
   - $n$ 为奇数：令 $t = \sin x$
   - $m,n$ 均为偶数：用倍角公式降幂

### 📚 常用三角积分公式

| 积分 | 结果 |
|------|------|
| $\int \sin^2 x dx$ | $\frac{x}{2} - \frac{\sin 2x}{4} + C$ |
| $\int \cos^2 x dx$ | $\frac{x}{2} + \frac{\sin 2x}{4} + C$ |
| $\int \tan x dx$ | $-\ln|\cos x| + C$ |
| $\int \cot x dx$ | $\ln|\sin x| + C$ |
| $\int \sec x dx$ | $\ln|\sec x + \tan x| + C$ |
| $\int \csc x dx$ | $\ln|\csc x - \cot x| + C$ |

### 🧮 示例演示
**例1**：求 $\int \sin^3 x \cos^2 x dx$
- $m=3$（奇数），令 $t = \cos x$，则 $dt = -\sin x dx$
- $\sin^2 x = 1 - t^2$，$\sin^3 x \cos^2 x = \sin^2 x \cdot \sin x \cdot \cos^2 x = (1-t^2) \cdot (-dt) \cdot t^2$
- $\int (1-t^2)t^2 (-dt) = -\int (t^2 - t^4) dt = -\left(\frac{t^3}{3} - \frac{t^5}{5}\right) + C$
- 回代：$-\frac{1}{3}\cos^3 x + \frac{1}{5}\cos^5 x + C$

**例2**：求 $\int \frac{dx}{\sin x}$
- 方法1：公式 $\int \csc x dx = \ln|\csc x - \cot x| + C$
- 方法2：万能代换 $t = \tan\frac{x}{2}$
  $\int \frac{dx}{\sin x} = \int \frac{1}{\frac{2t}{1+t^2}} \cdot \frac{2}{1+t^2} dt = \int \frac{dt}{t} = \ln|t| + C = \ln|\tan\frac{x}{2}| + C$
- 两种结果等价：$\ln|\tan\frac{x}{2}| = \ln|\csc x - \cot x|$

## 🔑 第四章答案与解析

### 基础题解析
1. **$\frac{1}{2}x^4 - \frac{3}{2}x^2 + \ln|x| + C$**
   - **解析**：$\int 2x^3 dx = \frac{1}{2}x^4$，$\int (-3x) dx = -\frac{3}{2}x^2$，$\int \frac{1}{x} dx = \ln|x|$

2. **$2e^{\sqrt{x}} + C$**
   - **解析**：令 $t = \sqrt{x}$，则 $x = t^2$，$dx = 2t dt$
     $\int \frac{e^t}{t} \cdot 2t dt = 2\int e^t dt = 2e^t + C = 2e^{\sqrt{x}} + C$

3. **$\frac{1}{3}x^3 \ln x - \frac{1}{9}x^3 + C$**
   - **解析**：令 $u = \ln x$，$dv = x^2 dx$ ⇒ $du = \frac{1}{x} dx$，$v = \frac{1}{3}x^3$
     $\int x^2 \ln x dx = \frac{1}{3}x^3 \ln x - \int \frac{1}{3}x^3 \cdot \frac{1}{x} dx = \frac{1}{3}x^3 \ln x - \frac{1}{3}\int x^2 dx = \frac{1}{3}x^3 \ln x - \frac{1}{9}x^3 + C$

4. **$\ln|x+1| + \ln|x+3| + C$**
   - **解析**：分母 $x^2+4x+3 = (x+1)(x+3)$
     设 $\frac{2x+3}{(x+1)(x+3)} = \frac{A}{x+1} + \frac{B}{x+3}$
     得 $A=1$，$B=1$
     $\int \left(\frac{1}{x+1} + \frac{1}{x+3}\right) dx = \ln|x+1| + \ln|x+3| + C$

5. **$\frac{1}{3}\sin^3 x - \frac{1}{5}\sin^5 x + C$**
   - **解析**：$\int \sin^2 x \cos^3 x dx = \int \sin^2 x \cos^2 x \cdot \cos x dx$
     令 $t = \sin x$，则 $dt = \cos x dx$，$\cos^2 x = 1 - t^2$
     $\int t^2 (1-t^2) dt = \int (t^2 - t^4) dt = \frac{1}{3}t^3 - \frac{1}{5}t^5 + C$

### 提高题解析
6. **$\frac{1}{13}e^{2x}(2\cos 3x + 3\sin 3x) + C$**
   - **解析**：令 $I = \int e^{2x} \cos 3x dx$
     分部积分两次得：
     $I = \frac{1}{2}e^{2x}\cos 3x + \frac{3}{2}\int e^{2x} \sin 3x dx$
     $\int e^{2x} \sin 3x dx = \frac{1}{2}e^{2x}\sin 3x - \frac{3}{2}\int e^{2x} \cos 3x dx = \frac{1}{2}e^{2x}\sin 3x - \frac{3}{2}I$
     代入解得：$I = \frac{1}{13}e^{2x}(2\cos 3x + 3\sin 3x) + C$

7. **$\frac{1}{2}x^2 - \frac{1}{2}\ln(x^2+1) + C$**
   - **解析**：$\frac{x^3}{x^2+1} = x - \frac{x}{x^2+1}$
     $\int x dx = \frac{1}{2}x^2$
     $\int \frac{x}{x^2+1} dx = \frac{1}{2}\int \frac{2x}{x^2+1} dx = \frac{1}{2}\ln(x^2+1)$
     结果：$\frac{1}{2}x^2 - \frac{1}{2}\ln(x^2+1) + C$

8. **$\ln\left|1+\tan\frac{x}{2}\right| + C$**
   - **解析**：令 $t = \tan\frac{x}{2}$，则 $\sin x = \frac{2t}{1+t^2}$，$\cos x = \frac{1-t^2}{1+t^2}$，$dx = \frac{2}{1+t^2} dt$
     代入：$\int \frac{1}{1+\frac{2t}{1+t^2}+\frac{1-t^2}{1+t^2}} \cdot \frac{2}{1+t^2} dt = \int \frac{2}{1+t^2+2t+1-t^2} dt = \int \frac{2}{2+2t} dt = \int \frac{1}{1+t} dt = \ln|1+t| + C$
