## 二重积分计算概述 *Content-[00:00]

- 二重积分是定积分的延伸（一个积分号变为两个积分号），积分区域用 `D` 表示，被积函数为二元函数 `f(x,y)`，微元为 `dx dy`。
- 考试中二重积分通常只考计算题，难度不高，属于“送分”题目。
- 解题关键是判断积分区域是 **X型** 还是 **Y型**，并套用对应的计算口诀。
- 视频从四个方面讲解二重积分：
  1. X型与Y型的基本计算
  2. 交换积分次序
  3. 极坐标系下的二重积分
  4. 含绝对值的二重积分（拔高内容）

## 二重积分的定义与表示 *Content-[01:00]

- 二重积分一般形式：`∬_D f(x,y) dx dy`。
- 积分区域 `D` 是点的集合，可用不等式描述。
- 复习定积分求面积时的 **X型**（上下型）与 **Y型**（左右型）概念。

## X型（上下型）二重积分的计算 *Content-[01:38]

- **X型**：积分区域上下由函数 `y = φ₁(x)`（下边界）和 `y = φ₂(x)`（上边界）围成，`x` 的取值范围为 `[a, b]`。
- 区域表示：`D = { (x,y) | a ≤ x ≤ b, φ₁(x) ≤ y ≤ φ₂(x) }`。
- **计算口诀**（X型）：
  - 后积先定线：先对 `x` 积分，写出 `x` 的上下限：`∫_a^b dx`
  - 线内画条线：在 `x` 区间内画一条垂直于 `x` 轴的线，从下向上穿过区域。
  - 先交写下线，后交写上线：先交到的函数为下限，后交到的函数为上限。
  - 展开为：`∫_a^b dx ∫_{φ₁(x)}^{φ₂(x)} f(x,y) dy`
- **计算顺序**：先计算内层积分（对 `y` 积分），结果再对外层 `x` 积分。内层积分计算时，`x` 视为常数。

### 例题1：X型计算 *Content-[06:10]

- **题目**：`D` 由 `y = x²` 和 `x = y²` 围成，计算 `∬_D (x+y) dx dy`。
- **步骤**：
  1. 画出草图：`y = x²` 开口向上，`x = y²`（即 `y = ±√x`）开口向右，围成第一象限的封闭区域。
  2. 判断类型：该区域既是 X型（上下型）也是 Y型（左右型）。先按 X型处理。
  3. 处理自变量：X型要求边界函数用 `x` 表示。上边界为 `y = √x`，下边界为 `y = x²`。`x` 范围：交点为 `(0,0)` 和 `(1,1)`，故 `x ∈ [0,1]`。
  4. 写出积分：
     ```
     ∬_D (x+y) dx dy = ∫_{0}^{1} dx ∫_{x²}^{√x} (x+y) dy
     ```
  5. 计算内层积分（对 `y`）：
     ```
     ∫_{x²}^{√x} (x+y) dy = [xy + ½ y²]_{x²}^{√x}
       = (x·√x + ½ (√x)²) - (x·x² + ½ (x²)²)
       = x^{3/2} + ½ x - x³ - ½ x⁴
     ```
  6. 外层积分（对 `x`）：
     ```
     ∫_{0}^{1} (x^{3/2} + ½ x - x³ - ½ x⁴) dx
       = [ (2/5)x^{5/2} + (1/4)x² - (1/4)x⁴ - (1/10)x⁵ ]_{0}^{1}
       = 2/5 + 1/4 - 1/4 - 1/10 = 2/5 - 1/10 = 3/10
     ```
- **结果**：`3/10`

## Y型（左右型）二重积分的计算 *Content-[12:26]

- **Y型**：积分区域左右由函数 `x = ψ₁(y)`（左边界）和 `x = ψ₂(y)`（右边界）围成，`y` 的取值范围为 `[c, d]`。
- 区域表示：`D = { (x,y) | c ≤ y ≤ d, ψ₁(y) ≤ x ≤ ψ₂(y) }`。
- **计算口诀**（Y型）：
  - 后积先定线：先对 `y` 积分，写出 `y` 的上下限：`∫_c^d dy`
  - 线内画条线：在 `y` 区间内画一条垂直于 `y` 轴的线，从左向右穿过区域。
  - 先交写下线，后交写上线：左边界为下限，右边界为上限。
  - 展开为：`∫_c^d dy ∫_{ψ₁(y)}^{ψ₂(y)} f(x,y) dx`
- **计算顺序**：先计算内层积分（对 `x` 积分），结果再对外层 `y` 积分。内层积分时，`y` 视为常数。

### 例题2：Y型计算（超越积分处理） *Content-[20:44]

- **题目**：`D` 由 `y = x` 和 `x = y²` 围成，计算 `∬_D (sin y)/y dx dy`。
- **分析**：若按 X型计算，内层积分对 `y` 会遇到 `∫ (sin y)/y dy`，该积分无法用初等函数表示（超越积分），故必须改用 Y型。
- **步骤**：
  1. 画出草图：`y = x` 为直线，`x = y²`（即 `y = √x` 和 `y = -√x`），围成区域（第一象限）。
  2. 判断类型：该区域也是 XY 型，但 X型不行，选 Y型。
  3. 处理自变量：Y型要求边界函数用 `y` 表示。右边界 `x = y`，左边界 `x = y²`。`y` 范围：`0 ≤ y ≤ 1`。
  4. 写出积分：
     ```
     ∬_D (sin y)/y dx dy = ∫_{0}^{1} dy ∫_{y²}^{y} (sin y)/y dx
     ```
  5. 计算内层积分（对 `x`）：
     ```
     ∫_{y²}^{y} (sin y)/y dx = (sin y)/y · (y - y²) = sin y - y sin y
     ```
  6. 外层积分（对 `y`）：
     ```
     ∫_{0}^{1} (sin y - y sin y) dy
     ```
     - 第一项 `∫ sin y dy = -cos y` 从 0 到 1，得 `-cos1 + 1`。
     - 第二项 `∫ y sin y dy` 使用分部积分：
       ```
       ∫ y sin y dy = -y cos y + ∫ cos y dy = -y cos y + sin y
       ```
       从 0 到 1 得 `(-1·cos1 + sin1) - (0 + 0) = -cos1 + sin1`。
     - 总和：`(-cos1 + 1) - (-cos1 + sin1) = 1 - sin1`
- **结果**：`1 - sin1`
- **教训**：当被积函数中某个变量出现复杂形式时，应优先考虑将该变量作为“后积”的变量（即先积另一个变量），从而简化计算。

## X型与Y型对比总结 *Content-[20:09]

| 类型 | 别名 | 后积变量 | 自变量 | 画线方向 | 口诀 |
|------|------|----------|--------|----------|------|
| X型 | 上下型 | X | X | 垂直于X轴（上下） | 后积先定线，线内画条线，先交写下线，后交写上线 |
| Y型 | 左右型 | Y | Y | 垂直于Y轴（左右） | 同上 |

- 关键：全程 X型就全程用 X，Y型就全程用 Y。口诀完全一致，只是方向不同。

## 交换积分次序 *Content-[28:28]

- **题型特征**：题目直接给出展开后的二重积分（如 `∫ dx ∫ dy` 的形式），要求交换积分次序（通常从 X型换为 Y型，或反之）。
- **解题步骤**：
  1. 从已知的积分上下限中，写出积分区域 `D` 的不等式描述。
     - 例如：`∫₀¹ dx ∫₀^{√x} dy` → `0 ≤ x ≤ 1, 0 ≤ y ≤ √x`
  2. 画出该区域图形（通常涉及直线、曲线）。
  3. 判断区域是否能表示为另一种类型（Y型或X型）。
  4. 按新的类型写出积分表达式，注意边界函数的自变量转换。

### 例题3：交换积分次序 *Content-[28:28]

- **原积分**：
  ```
  ∫₀¹ dx ∫₀^{√x} f(x,y) dy + ∫₁² dx ∫₀^{2-x} f(x,y) dy
  ```
- **步骤**：
  - 写出不等式：
    - 第一块：`0 ≤ x ≤ 1, 0 ≤ y ≤ √x` → 即 `y = √x` 下方区域。
    - 第二块：`1 ≤ x ≤ 2, 0 ≤ y ≤ 2-x` → 即 `y = 2-x` 下方区域。
  - 画图：`y = √x` 为抛物线的一部分（第一象限），`y = 2-x` 为直线，两曲线交点为 `(1,1)`。
  - 整个区域被 `x=1` 分成两部分，合并后整体区域可视为 Y型。
  - Y型描述：`0 ≤ y ≤ 1`，左边界 `x = y²`（由 `y = √x` 反解得），右边界 `x = 2-y`（由 `y = 2-x` 反解得）。
  - 交换次序后的积分：
    ```
    ∫₀¹ dy ∫_{y²}^{2-y} f(x,y) dx
    ```

### 例题4：含圆域的交换积分次序 *Content-[34:12]

- **原积分**：
  ```
  ∫₀¹ dx ∫₀^{1-√(1-x²)} f(x,y) dy + ∫₁^{√2} dx ∫₀^{√(2-x²)} f(x,y) dy
  ```
- **步骤**：
  - 写出不等式：
    - 第一块：`0 ≤ x ≤ 1, 0 ≤ y ≤ 1-√(1-x²)` → 该曲线 `y = 1-√(1-x²)` 是下半圆的方程（圆心 (0,1)，半径 1）。
    - 第二块：`1 ≤ x ≤ √2, 0 ≤ y ≤ √(2-x²)` → 该曲线 `y = √(2-x²)` 是上半圆的方程（圆心 (0,0)，半径 √2）。
  - 画图：两圆交点为 `(1,1)`。整个区域是由两段圆弧围成的曲边梯形。
  - 该区域可视为 Y型：`0 ≤ y ≤ 1`，左边界由圆 `(x)² + (y-1)² = 1` 解出 `x = √(1 - (y-1)²)`（取正），右边界由圆 `x² + y² = 2` 解出 `x = √(2 - y²)`（取正）。
  - 交换次序后的积分：
    ```
    ∫₀¹ dy ∫_{√(1-(y-1)²)}^{√(2-y²)} f(x,y) dx
    ```

## 极坐标系下的二重积分 *Content-[41:56]

- **适用条件**：积分区域含有圆（圆域、半圆、扇形等），或被积函数含有 `x² + y²` 的形式。
- **极坐标变换**：
  ```
  x = r cos θ
  y = r sin θ
  dx dy = r dr dθ
  ```
- **换圆三件套**：积分区域、被积函数、微元（尤其是 `dx dy` 变为 `r dr dθ`，易漏掉 `r`）。
- **极坐标系下积分区域的表示**：
  - `θ` 表示极角（从 x 轴正方向逆时针旋转的角度）。
  - `r` 表示极径（点到原点的距离）。
  - 积分区域 `D` 通常可描述为：`θ ∈ [α, β]`，`r ∈ [r₁(θ), r₂(θ)]`。
- **计算口诀（极坐标下）**：
  - 后积先定线：先定 `θ` 的上下限（角度范围）。
  - 线内画条线：从极点出发向外画射线，先交的边界为 `r` 的下限，后交的为上限。
  - 注意：`dx dy` 替换为 `r dr dθ`。
  - 被积函数：`f(x,y)` 变为 `f(r cos θ, r sin θ)`。

### 极坐标系基本概念 *Content-[45:55]

- **极点**：直角坐标系的原点。
- **极轴**：x 轴正半轴。
- **极角 θ**：从极轴逆时针旋转的角度。
- **极径 r**：点到极点的距离。
- 例：直角坐标点 `(1,1)` 在极坐标下可表示为 `r = √2, θ = π/4`。

### 极坐标系下二重积分的计算步骤 *Content-[50:38]

- 根据积分区域形状，确定 `θ` 的范围（通常由区域边界切线决定）。
- 然后确定 `r` 的边界函数 `r₁(θ)` 和 `r₂(θ)`。
- 写出积分：
  ```
  ∬_D f(x,y) dx dy = ∫_{θ=α}^{β} dθ ∫_{r=r₁(θ)}^{r₂(θ)} f(r cosθ, r sinθ) · r dr
  ```
- **重要提醒**：极坐标换元后，`dx dy` 必须变成 `r dr dθ`，不要漏掉因子 `r`。
