## 1. 基本概念与定义 *Content-[00:18]

- **空间曲线**：给定一条空间曲线，过其上一点可作一条**切线**（沿曲线方向）。
- **法平面**：过该点且与切线垂直的平面称为法平面。
- 关键区分：曲线对应“切线”与“法平面”；曲面则对应“切平面”与“法线”，不要混淆。

---

## 2. 参数方程形式下的切线与法平面 *Content-[02:00]

设空间曲线的参数方程为：
\[
\begin{cases}
x = x(t) \\
y = y(t) \\
z = z(t)
\end{cases}
\]
记向量函数 \(\vec{f}(t) = (x(t), y(t), z(t))\)。

- **切向量**（方向向量）：
  \[
  \vec{T} = \vec{f}'(t_0) = \big( x'(t_0),\; y'(t_0),\; z'(t_0) \big)
  \]

- **切线方程**（点向式）：
  \[
  \frac{x - x_0}{x'(t_0)} = \frac{y - y_0}{y'(t_0)} = \frac{z - z_0}{z'(t_0)}
  \]
  其中 \((x_0, y_0, z_0)\) 为切点。

- **法平面方程**（点法式）：
  \[
  x'(t_0)(x - x_0) + y'(t_0)(y - y_0) + z'(t_0)(z - z_0) = 0
  \]
  法向量即切向量 \(\vec{T}\)。

## 3. 例题：参数方程求切线和法平面 *Content-[05:43]

**题目**：曲线 \(x = t,\; y = t^2,\; z = t^3\) 在点 \((1,1,1)\) 处的切线和法平面。

**解**：
- 求导：\(x' = 1,\; y' = 2t,\; z' = 3t^2\)。
- 点 \((1,1,1)\) 对应 \(t=1\)，代入得切向量 \(\vec{T} = (1, 2, 3)\)。
- **切线方程**（点向式）：
  \[
  \frac{x-1}{1} = \frac{y-1}{2} = \frac{z-1}{3}
  \]
- **法平面方程**（点法式）：
  \[
  1\cdot(x-1) + 2\cdot(y-1) + 3\cdot(z-1) = 0
  \]
  化简得 \(x + 2y + 3z = 6\)。

## 4. 特殊情况：y 和 z 是 x 的函数 *Content-[08:22]

当曲线表示为 \(y = y(x),\; z = z(x)\) 时，可视为参数方程 \(x = x\)（视 x 为参数）。此时：
- 切向量为 \(\big(1,\; y'(x_0),\; z'(x_0)\big)\)。
- **切线方程**：
  \[
  \frac{x-x_0}{1} = \frac{y-y_0}{y'(x_0)} = \frac{z-z_0}{z'(x_0)}
  \]
- **法平面方程**：
  \[
  1\cdot(x-x_0) + y'(x_0)(y-y_0) + z'(x_0)(z-z_0) = 0
  \]

> 此形式可由一般参数方程令 \(t = x\) 直接导出，无需额外记忆。

## 5. 两个曲面相交形成的曲线 *Content-[10:09]

当空间曲线由两个曲面方程 \(F(x,y,z)=0\) 和 \(G(x,y,z)=0\) 的交线给出时，求切线和法平面的常用方法（避免背诵复杂理论）：

1. 将 \(y\) 和 \(z\) 视为 \(x\) 的函数，对两个方程两边分别对 \(x\) 求导（注意 \(y,z\) 是复合函数）。
2. 得到关于 \(\frac{dy}{dx}\) 和 \(\frac{dz}{dx}\) 的线性方程组。
3. 解出 \(\frac{dy}{dx}\) 和 \(\frac{dz}{dx}\) 在切点处的值。
4. 切向量为 \(\big(1,\; \frac{dy}{dx},\; \frac{dz}{dx}\big)\)。
5. 代用点向式和点法式得到切线方程和法平面方程。

## 6. 例题：球面与平面相交形成的圆 *Content-[11:12]

**题目**：曲线由
\[
\begin{cases}
x^2 + y^2 + z^2 = 6 \\
x + y + z = 0
\end{cases}
\]
给出，求在点 \((1, -2, 1)\) 处的切线和法平面。

**解**：
- 将 \(y, z\) 视为 \(x\) 的函数，对两个方程两边对 \(x\) 求导：
  \[
  \begin{cases}
  2x + 2y\frac{dy}{dx} + 2z\frac{dz}{dx} = 0 \\
  1 + \frac{dy}{dx} + \frac{dz}{dx} = 0
  \end{cases}
  \]
  化简得：
  \[
  \begin{cases}
  y\frac{dy}{dx} + z\frac{dz}{dx} = -x \\
  \frac{dy}{dx} + \frac{dz}{dx} = -1
  \end{cases}
  \]

- 代入点 \((1, -2, 1)\)：
  \[
  \begin{cases}
  -2\frac{dy}{dx} + 1\cdot\frac{dz}{dx} = -1 \\
  \frac{dy}{dx} + \frac{dz}{dx} = -1
  \end{cases}
  \]
  解得：\(\frac{dy}{dx} = 0,\; \frac{dz}{dx} = -1\)。

- 切向量：\(\big(1,\; 0,\; -1\big)\)。
- **切线方程**（点向式）：
  \[
  \frac{x-1}{1} = \frac{y+2}{0} = \frac{z-1}{-1}
  \]
- **法平面方程**（点法式）：
  \[
  1\cdot(x-1) + 0\cdot(y+2) + (-1)\cdot(z-1) = 0
  \]
  化简得 \(x - z = 0\)。
