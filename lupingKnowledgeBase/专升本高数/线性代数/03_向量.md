## 3.1 向量的概念与运算

**内容待补充**：向量定义、线性运算、数量积、向量积。

### 向量定义
- 既有大小又有方向的量
- $n$维向量：$(a_1, a_2, \dots, a_n)$

### 向量运算
1. **加法**：平行四边形法则
2. **数乘**：$k\vec{a}$，长度乘$k$，方向相同或相反
3. **数量积**：$\vec{a} \cdot \vec{b} = |\vec{a}||\vec{b}|\cos\theta$
4. **向量积**：$\vec{a} \times \vec{b}$（三维空间）

### 向量性质
1. $\vec{a} + \vec{b} = \vec{b} + \vec{a}$
2. $(\vec{a} + \vec{b}) + \vec{c} = \vec{a} + (\vec{b} + \vec{c})$
3. $k(\vec{a} + \vec{b}) = k\vec{a} + k\vec{b}$

## 3.2 向量组的线性相关性

**内容待补充**：线性相关与无关定义、判断方法。

### 线性相关定义
向量组 $\vec{a}_1, \vec{a}_2, \dots, \vec{a}_m$ 线性相关 ⇔ 存在不全为0的数 $k_1, k_2, \dots, k_m$，使 $k_1\vec{a}_1 + k_2\vec{a}_2 + \cdots + k_m\vec{a}_m = \vec{0}$

### 线性无关定义
向量组 $\vec{a}_1, \vec{a}_2, \dots, \vec{a}_m$ 线性无关 ⇔ 若 $k_1\vec{a}_1 + k_2\vec{a}_2 + \cdots + k_m\vec{a}_m = \vec{0}$，则必有 $k_1 = k_2 = \cdots = k_m = 0$

### 判断方法
1. **定义法**：解齐次线性方程组
2. **秩的判断法**：向量组构成的矩阵秩 < 向量个数 ⇔ 线性相关
3. **特殊情形**：
   - 单个非零向量线性无关
   - 包含零向量的向量组线性相关
   - $n+1$个$n$维向量必线性相关

## 3.3 向量组的秩

**内容待补充**：极大无关组、向量组的秩。

### 极大无关组
向量组$S$的一个部分组如果满足：
1. 线性无关
2. $S$中任意向量可由该部分组线性表示
则称该部分组为$S$的一个极大线性无关组。

### 向量组的秩
向量组的极大无关组所含向量的个数，记作$r(S)$。

### 性质
1. 向量组的秩等于其构成矩阵的秩
2. 等价向量组有相同的秩
3. $r(S) \leq \min\{$向量个数，向量维数$\}$

### 求极大无关组方法
1. 将向量组按列排成矩阵
2. 用初等行变换化为行阶梯形
3. 主元所在列对应的原向量构成极大无关组

## 3.4 向量空间

**内容待补充**：向量空间定义、子空间、基与维数。

### 向量空间定义
设$V$是非空集合，$P$是数域，若在$V$中定义了加法和数乘运算，且满足8条公理，则称$V$是数域$P$上的向量空间。

### 子空间
$V$的非空子集$W$如果对加法和数乘封闭，则$W$是$V$的子空间。

### 基与维数
- **基**：向量空间中极大线性无关组
- **维数**：基所含向量的个数，记作$\dim V$
- **坐标**：向量在基下的线性表示系数

### 常见向量空间
1. $R^n$：$n$维实向量空间
2. $P_n[x]$：次数不超过$n$的多项式空间
3. $M_{m\times n}$：$m\times n$矩阵空间
