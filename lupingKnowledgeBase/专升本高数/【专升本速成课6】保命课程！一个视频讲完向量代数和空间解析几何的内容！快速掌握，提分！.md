## 1. 向量运算 [原片 @ 00:00](https://www.bilibili.com/video/BV1nRFfzDEHJ?t=0)
### 1.1 数量积（点积）
*   **定义**：表示一个**数**。
*   **模长计算**：向量 $\vec{a} = (x_1, y_1, z_1)$ 的模长为 $|\vec{a}| = \sqrt{x_1^2 + y_1^2 + z_1^2}$。
*   **点积计算**：$\vec{a} \cdot \vec{b} = x_1x_2 + y_1y_2 + z_1z_2$。
*   **几何意义**：$\vec{a} \cdot \vec{b} = |\vec{a}| |\vec{b}| \cos \theta$，其中 $\theta$ 为两向量夹角。
*   **垂直判定**：两向量垂直 $\Leftrightarrow$ $\vec{a} \cdot \vec{b} = 0$。

### 1.2 向量积（叉积）
*   **定义**：表示一个**向量**。结果向量 $\vec{c} = \vec{a} \times \vec{b}$ 同时垂直于 $\vec{a}$ 和 $\vec{b}$，方向遵循右手定则。
*   **大小计算**：$|\vec{a} \times \vec{b}| = |\vec{a}| |\vec{b}| \sin \theta$。
*   **与点积的核心区别**：点积结果是一个数（含 $\cos \theta$），叉积结果是一个向量（含 $\sin \theta$）。
*   **行列式计算法**：
    $$\vec{a} \times \vec{b} =
    \begin{vmatrix}
    \vec{i} & \vec{j} & \vec{k} \\
    x_1 & y_1 & z_1 \\
    x_2 & y_2 & z_2
    \end{vmatrix} = (y_1z_2 - y_2z_1)\vec{i} - (x_1z_2 - x_2z_1)\vec{j} + (x_1y_2 - x_2y_1)\vec{k}$$
    *   二阶行列式计算口诀：“主对角线乘积 - 副对角线乘积”。
*   **重要性质**：
    1.  $\vec{a} \times \vec{a} = \vec{0}$。
    2.  $k(\vec{a} \times \vec{b}) = (k\vec{a}) \times \vec{b} = \vec{a} \times (k\vec{b})$。
    3.  $\vec{a} \times \vec{b} = -(\vec{b} \times \vec{a})$（反交换律）。
    4.  两向量平行 $\Leftrightarrow$ $\vec{a} \times \vec{b} = \vec{0}$ $\Leftrightarrow$ $\frac{x_1}{x_2} = \frac{y_1}{y_2} = \frac{z_1}{z_2}$（对应分量成比例）。

### 1.3 方向角与方向余弦
*   **方向角**：向量 $\vec{v} = (x, y, z)$ 与 $x, y, z$ 轴正方向的夹角 $\alpha, \beta, \gamma$。
*   **方向余弦**：
    $$\cos \alpha = \frac{x}{|\vec{v}|}, \quad \cos \beta = \frac{y}{|\vec{v}|}, \quad \cos \gamma = \frac{z}{|\vec{v}|}$$
*   **重要关系**：$\cos^2 \alpha + \cos^2 \beta + \cos^2 \gamma = 1$。
*   **单位向量**：$\vec{v}^0 = \frac{\vec{v}}{|\vec{v}|} = (\cos \alpha, \cos \beta, \cos \gamma)$。**单位向量的坐标就是其方向余弦**。

### 1.4 典型例题与技巧
1.  **计算叉积模长**：已知 $|\vec{a}|=6, |\vec{b}|=3$，夹角 $\theta = \frac{\pi}{6}$，则 $|\vec{a} \times \vec{b}| = |\vec{a}||\vec{b}|\sin\theta = 6 \times 3 \times \frac{1}{2} = 9$。
2.  **化简含叉积的表达式**：计算 $(2\vec{a} + \vec{b}) \times (\vec{a} - 2\vec{b})$。
    *   利用叉积分配律和性质展开：$2\vec{a} \times \vec{a} - 4\vec{a} \times \vec{b} + \vec{b} \times \vec{a} - 2\vec{b} \times \vec{b}$。
    *   $\vec{a} \times \vec{a} = \vec{b} \times \vec{b} = \vec{0}$，且 $\vec{b} \times \vec{a} = -\vec{a} \times \vec{b}$。
    *   原式 $= -4\vec{a} \times \vec{b} - \vec{a} \times \vec{b} = -5\vec{a} \times \vec{b}$。
3.  **求常数与向量叉积的模长**：$|k(\vec{a} \times \vec{b})| = |k| \cdot |\vec{a} \times \vec{b}|$。
4.  **判断方向角**：若某向量 $\vec{v} = (0, 1, 1)$，则其 $x$ 轴方向余弦 $\cos \alpha = 0$，故 $\alpha = \frac{\pi}{2}$。利用 $\cos^2 \alpha + \cos^2 \beta + \cos^2 \gamma = 1$ 可快速排除错误选项。

## 2. 平面方程 [原片 @ 15:29](https://www.bilibili.com/video/BV1nRFfzDEHJ?t=929)
### 2.1 点法式方程
*   **已知条件**：平面上一点 $M_0(x_0, y_0, z_0)$ 及平面的法向量 $\vec{n} = (A, B, C)$。
*   **方程**：$A(x - x_0) + B(y - y_0) + C(z - z_0) = 0$。
*   **推导**：设平面上动点 $M(x, y, z)$，则向量 $\overrightarrow{M_0M} = (x-x_0, y-y_0, z-z_0)$ 在平面内。法向量 $\vec{n}$ 与平面内任何向量垂直，故 $\vec{n} \cdot \overrightarrow{M_0M} = 0$。

### 2.2 一般式方程
*   **方程**：$Ax + By + Cz + D = 0$。
*   **与点法式关系**：将点法式展开并令常数项为 $D$ 即得一般式。
*   **关键结论**：**一般式方程中 $x, y, z$ 的系数 $(A, B, C)$ 就是该平面的一个法向量 $\vec{n}$**。

### 2.3 求平面方程典型例题
*   **题目**：求过点 $A(1,0,2)$ 且与两平面 $\Pi_1: x - y + z = 1$ 和 $\Pi_2: x - z = 3$ 都垂直的平面方程。
*   **解题步骤**：
    1.  已知点 $A(1,0,2)$。
    2.  所求平面 $\Pi$ 与 $\Pi_1, \Pi_2$ 都垂直 $\Rightarrow$ $\Pi$ 的法向量 $\vec{n}$ 与 $\Pi_1, \Pi_2$ 的法向量 $\vec{n_1}, \vec{n_2}$ 都垂直。
    3.  取 $\vec{n} = \vec{n_1} \times \vec{n_2}$。
        *   $\vec{n_1} = (1, -1, 1)$， $\vec{n_2} = (1, 0, -1)$。
        *   $\vec{n} = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ 1 & -1 & 1 \\ 1 & 0 & -1 \end{vmatrix} = (1, 2, 1)$。
    4.  代入点法式：$1 \cdot (x-1) + 2 \cdot (y-0) + 1 \cdot (z-2) = 0$。
    5.  化简为一般式：$x + 2y + z - 3 = 0$。

## 3. 直线方程 [原片 @ 26:14](https://www.bilibili.com/video/BV1nRFfzDEHJ?t=1574)
### 3.1 一般式（交面式）
*   **方程**：$\begin{cases} A_1x + B_1y + C_1z + D_1 = 0 \\ A_2x + B_2y + C_2z + D_2 = 0 \end{cases}$，表示两个平面的交线。
*   **方向向量求法**：直线的方向向量 $\vec{s}$ 同时垂直于两个平面的法向量 $\vec{n_1} = (A_1, B_1, C_1)$ 和 $\vec{n_2} = (A_2, B_2, C_2)$。
    *   **结论**：$\vec{s} = \vec{n_1} \times \vec{n_2}$。

### 3.2 点向式（对称式）
*   **已知条件**：直线上一点 $M_0(x_0, y_0, z_0)$ 及方向向量 $\vec{s} = (m, n, p)$。
*   **方程**：$\frac{x - x_0}{m} = \frac{y - y_0}{n} = \frac{z - z_0}{p}$。
*   **推导**：设直线上动点 $M(x, y, z)$，则 $\overrightarrow{M_0M} = (x-x_0, y-y_0, z-z_0)$ 与 $\vec{s}$ 平行，对应坐标成比例。

### 3.3 参数式
*   **方程**：令点向式等于参数 $t$，即 $\begin{cases} x = x_0 + mt \\ y = y_0 + nt \\ z = z_0 + pt \end{cases}$。
*   **关键**：**参数方程中 $t$ 的系数 $(m, n, p)$ 就是方向向量 $\vec{s}$**。

### 3.4 求直线方程典型例题
*   **题型一**：求过点 $M(1,2,3)$ 且与平面 $2x+3y-z=5$ 平行，与直线 $L: \frac{x-1}{1} = \frac{y+1}{3} = \frac{z}{4}$ 垂直的直线方程。
    *   **思路**：设所求直线方向向量为 $\vec{s}$。
        1.  与平面平行 $\Rightarrow$ $\vec{s} \perp$ 平面法向量 $\vec{n_p} = (2,3,-1)$。
        2.  与直线 $L$ 垂直 $\Rightarrow$ $\vec{s} \perp$ $L$ 的方向向量 $\vec{s_L} = (1,3,4)$。
    *   **解法**：取 $\vec{s} = \vec{n_p} \times \vec{s_L}$，计算后得到 $\vec{s}$，再代入点 $M$ 用点向式写出方程。
*   **题型二**：求过点 $P$ 且与已知直线平行的直线方程。
    *   **思路**：直接取已知直线的方向向量作为所求直线的方向向量，再用点向式。
*   **点到平面的距离公式**：
    点 $P(x_0, y_0, z_0)$ 到平面 $Ax+By+Cz+D=0$ 的距离为：
    $$d = \frac{|Ax_0 + By_0 + Cz_0 + D|}{\sqrt{A^2 + B^2 + C^2}}$$

## 4. 二次曲面 [原片 @ 40:58](https://www.bilibili.com/video/BV1nRFfzDEHJ?t=2458)
### 4.1 核心特征与分类
考试主要考察根据方程形式判断曲面的大类，而非具体名称（如椭圆抛物面、双曲抛物面等）。

1.  **球面**：$(x-x_0)^2 + (y-y_0)^2 + (z-z_0)^2 = R^2$。
    *   **特征**：$x^2, y^2, z^2$ 系数均为正且相等（可化为1），无交叉项。
2.  **椭球面**：$\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1$。
    *   **特征**：$x^2, y^2, z^2$ 项系数均为正，右边为1。
3.  **双曲面**：
    *   **单叶双曲面**：$\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = 1$。
        *   **特征**：$x^2, y^2, z^2$ 项系数**两正一负**，右边为1。
    *   **双叶双曲面**：$-\frac{x^2}{a^2} - \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1$ 或 $\frac{x^2}{a^2} - \frac{y^2}{b^2} - \frac{z^2}{c^2} = 1$。
        *   **特征**：$x^2, y^2, z^2$ 项系数**两负一正**，右边为1。
4.  **椭圆锥面**：$\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = 0$。
    *   **特征**：$x^2, y^2, z^2$ 项系数两正一负，**右边为0**。可化为 $z = \pm c\sqrt{\frac{x^2}{a^2} + \frac{y^2}{b^2}}$。
5.  **抛物面**：
    *   **椭圆抛物面**：$\frac{x^2}{a^2} + \frac{y^2}{b^2} = z$
    *   **双曲抛物面（马鞍面）**：$\frac{x^2}{a^2} - \frac{y^2}{b^2} = z$
    *   **核心特征**：方程中**一个变量是一次项**，另外两个变量是二次项。即 **“二次，二次，一次”** 的形式。这是与前面所有曲面的根本区别。
6.  **柱面**：方程中**缺少一个变量**，如 $F(x, y)=0$ 表示母线平行于 $z$ 轴的柱面。
    *   **特征**：方程只含两个变量。

### 4.2 解题技巧
*   看方程中 $x^2, y^2, z^2$ 项的**符号**和**等号右边的常数**。
*   抓住 **“二次，二次，一次”** 是抛物面的标志。
*   抓住**缺少变量**是柱面的标志。

## 5. AI 总结
本视频系统梳理了向量代数与空间解析几何的核心考点，内容紧凑，直击专升本考试要点。
1.  **向量运算**：重点区分**点积**（结果为数，用于求夹角、判垂直）和**叉积**（结果为向量，用于求法向量、判平行、求面积）。方向余弦及其平方和为1是常考性质。
2.  **平面方程**：核心是 **“点法式”** ，关键在于根据条件（如与已知平面垂直、平行）求出法向量，通常需要用到**叉积**运算。
3.  **直线方程**：核心是 **“点向式”** ，关键在于求方向向量。若直线由两平面交线给出，则方向向量为两平面法向量的**叉积**。直线与平面、直线与直线的位置关系（平行、垂直）都转化为方向向量与法向量的垂直关系。
4.  **二次曲面**：无需死记具体图形，掌握**分类特征**即可应试：根据方程中二次项系数的**正负**和**等号右侧常数**区分球面、椭球面、双曲面、锥面；根据是否出现**一次项**识别抛物面；根据是否**缺元**识别柱面。
整体而言，这部分题目多为“送分题”，熟练掌握上述公式、性质及求法向量/方向向量的叉积方法，即可快速得分。
