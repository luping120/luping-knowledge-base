## 一、向量代数（基本运算） [原片 @ 00:14](https://www.bilibili.com/video/BV1nRFfzDEHJ?t=14)
### 1. **向量的模长（数量积/点积）**
*   表示一个**数**。
*   计算公式（模长）：$|\vec{a}| = \sqrt{x_1^2 + y_1^2 + z_1^2}$
*   点积（Dot Product）计算：$\vec{a} \cdot \vec{b} = x_1x_2 + y_1y_2 + z_1z_2$
*   **向量垂直的充要条件**：$\vec{a} \perp \vec{b} \Leftrightarrow \vec{a} \cdot \vec{b} = 0$ (夹角 $\theta = 90^\circ, \cos \theta = 0$)

### 2. **向量积（叉积）**
*   表示一个**向量**。
*   几何意义：结果向量 **同时垂直于** $\vec{a}$ 和 $\vec{b}$，方向遵循右手法则。
*   大小公式：$|\vec{a} \times \vec{b}| = |\vec{a}| |\vec{b}| \sin \theta$
*   运算方法（三阶行列式）：
    $$
    \vec{a} \times \vec{b} =
    \begin{vmatrix}
    \vec{i} & \vec{j} & \vec{k} \\
    x_1 & y_1 & z_1 \\
    x_2 & y_2 & z_2
    \end{vmatrix}
    $$

### 3. **重要性质**
*   $\vec{a} \times \vec{a} = \vec{0}$ （$\sin 0 = 0$）
*   $\vec{a} \times \vec{b} = - (\vec{b} \times \vec{a})$ （交换律不成立，反交换）
*   **向量平行的充要条件**：$\vec{a} \parallel \vec{b} \Leftrightarrow \vec{a} \times \vec{b} = \vec{0}$ （$\theta = 0^\circ \text{或} 180^\circ, \sin \theta = 0$）
    *   另一种表述：对应坐标成比例，即 $\frac{x_1}{x_2} = \frac{y_1}{y_2} = \frac{z_1}{z_2}$

### 4. **方向角与方向余弦** [原片 @ 08:27](https://www.bilibili.com/video/BV1nRFfzDEHJ?t=507)
设向量 $\vec{v} = (x, y, z)$，与坐标轴正向的夹角 $\alpha, \beta, \gamma$ 称为**方向角**。
*   **方向余弦**：
    $\cos \alpha = \frac{x}{|\vec{v}|}, \quad \cos \beta = \frac{y}{|\vec{v}|}, \quad \cos \gamma = \frac{z}{|\vec{v}|}$
*   **重要性质**：$\cos^2 \alpha + \cos^2 \beta + \cos^2 \gamma = 1$
*   **单位向量**：$\vec{v}^0 = (\cos \alpha, \cos \beta, \cos \gamma)$，即方向余弦的坐标就是其单位向量的坐标。

## 二、平面方程 [原片 @ 15:30](https://www.bilibili.com/video/BV1nRFfzDEHJ?t=930)
### 1. **点法式**
已知平面上一点 $M_0(x_0, y_0, z_0)$ 和法向量 $\vec{n} = (A, B, C)$，则平面方程为：
$$A(x - x_0) + B(y - y_0) + C(z - z_0) = 0$$

### 2. **一般式**
由点法式展开得到：
$$Ax + By + Cz + D = 0$$
其中，$(A, B, C)$ 即为平面的**法向量** $\vec{n}$。

### 3. **截距式**
平面在坐标轴上的截距分别为 $a, b, c$，则方程为：
$$\frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1$$
### 4. **例题：求与两平面均垂直的平面**
**思路**：所求平面的法向量 $\vec{n}$ 应同时垂直于两已知平面的法向量 $\vec{n}_1$ 和 $\vec{n}_2$。
因此，取 $\vec{n} = \vec{n}_1 \times \vec{n}_2$。
再利用点法式即可求得平面方程。

## 三、直线方程 [原片 @ 26:15](https://www.bilibili.com/video/BV1nRFfzDEHJ?t=1575)
### 1. **一般式（交面式）**
表示为两个平面的交线：
$$
\begin{cases}
A_1x + B_1y + C_1z + D_1 = 0 \\
A_2x + B_2y + C_2z + D_2 = 0
\end{cases}
$$
*   **关键**：直线的**方向向量 $\vec{s}$** 同时垂直于两平面的法向量 $\vec{n}_1, \vec{n}_2$。
*   **求法**：$\vec{s} = \vec{n}_1 \times \vec{n}_2$

### 2. **点向式（对称式）**
已知直线上一点 $M_0(x_0, y_0, z_0)$ 和方向向量 $\vec{s} = (l, m, n)$，则方程为：
$$\frac{x - x_0}{l} = \frac{y - y_0}{m} = \frac{z - z_0}{n}$$

### 3. **参数式**
由点向式引入参数 $t$ 得到：
$$
\begin{cases}
x = x_0 + lt \\
y = y_0 + mt \\
z = z_0 + nt
\end{cases}
$$
**注意**：参数式方程中，$t$ 前的系数 $(l, m, n)$ 即为方向向量。

### 4. **点到平面的距离公式**
点 $P(x_0, y_0, z_0)$ 到平面 $Ax + By + Cz + D = 0$ 的距离为：
$$d = \frac{|Ax_0 + By_0 + Cz_0 + D|}{\sqrt{A^2 + B^2 + C^2}}$$

## 四、二次曲面 [原片 @ 41:00](https://www.bilibili.com/video/BV1nRFfzDEHJ?t=2460)
识别二次曲面的关键是看方程的标准形式。
| 曲面类型 | 标准方程 (右侧=1) | 核心特征 | 常见图形 |
| :--- | :--- | :--- | :--- |
| **椭球面** | $\dfrac{x^2}{a^2} + \dfrac{y^2}{b^2} + \dfrac{z^2}{c^2} = 1$ | “+”号，全二次项，有系数 | 橄榄球/椭球 |
| **单叶双曲面** | $\dfrac{x^2}{a^2} + \dfrac{y^2}{b^2} - \dfrac{z^2}{c^2} = 1$ | 两正一负，全二次项 | 冷却塔状 |
| **双叶双曲面** | $\dfrac{x^2}{a^2} - \dfrac{y^2}{b^2} - \dfrac{z^2}{c^2} = 1$ | 一正两负，全二次项 | 两个分离的“碗” |
| **椭圆抛物面** | $\dfrac{x^2}{a^2} + \dfrac{y^2}{b^2} = z$ | **二，二，一**（次数） | 碗状/卫星天线 |
| **双曲抛物面** | $\dfrac{x^2}{a^2} - \dfrac{y^2}{b^2} = z$ | **二，二，一**（次数） | 马鞍面 |
| **锥面** | $\dfrac{x^2}{a^2} + \dfrac{y^2}{b^2} - \dfrac{z^2}{c^2} = 0$ | 全二次项，**右侧为0** | 圆锥/椭圆锥 |
| **柱面** | 缺一个变量，如 $F(x, y)=0$ | 方程中**缺少一个变量** | 平行于缺省轴的柱体 |

**考试要点**：
1.  看到“二，二，一”次（即两个变量二次，一个变量一次），就是**抛物面**。
2.  看到方程**右侧为0**且全是二次项，就是**锥面**。
3.  看到方程中**缺少一个变量**（如无 $z$），就是**柱面**。
