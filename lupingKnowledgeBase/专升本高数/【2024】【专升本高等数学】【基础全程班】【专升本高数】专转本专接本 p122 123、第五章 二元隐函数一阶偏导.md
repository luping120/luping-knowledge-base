## 1. 什么是二元隐函数 *Content-[00:00]

- 二元显函数的形式为：`z = f(x, y)`
- 只要不是这种形式的二元函数，就是 **二元隐函数**。
- 一般教材中描述为：由方程 `F(x, y, z) = 0` 所确定的函数。

## 2. 二元隐函数一阶偏导的求解方法 *Content-[00:50]

使用 **公式法**，主要步骤：

1. **移项并构造 `F(x, y, z)`**  
   将题干方程所有项移到一边，令左边等于 `F(x, y, z)`（例如：`F = 左边 - 右边`）。

2. **求偏导**  
   分别求出 `F` 对 `x`、`y`、`z` 的偏导：`F_x`、`F_y`、`F_z`。

3. **套用公式**  
   $$
   \frac{\partial z}{\partial x} = -\frac{F_x}{F_z}, \quad \frac{\partial z}{\partial y} = -\frac{F_y}{F_z}
   $$

> 注：此公式由复合函数求导推导而来，只需记住公式即可。

## 3. 例题一：基础求偏导 *Content-[02:58]

**题目**：  
函数 `z = z(x, y)` 由方程 `z + e^z = xy` 所确定，求 `∂z/∂x` 和 `∂z/∂y`。

**解答**：

1. 移项构造  
   $$
   F(x, y, z) = z + e^z - xy
   $$

2. 求偏导  
   - `F_x = -y`  
   - `F_y = -x`  
   - `F_z = 1 + e^z`

3. 套公式  
   $$
   \frac{\partial z}{\partial x} = -\frac{F_x}{F_z} = \frac{y}{1 + e^z}
   $$
   $$
   \frac{\partial z}{\partial y} = -\frac{F_y}{F_z} = \frac{x}{1 + e^z}
   $$

## 4. 例题二：求全微分（真题） *Content-[05:34]

**题目**（2015年 4分题）：  
由方程 `xyz + √(x² + y² + z²) = √2` 所确定的函数 `z = z(x, y)`，求在点 `(1, 0, -1)` 处的全微分 `dz`。

**解答**：

1. 移项构造  
   $$
   F(x, y, z) = xyz + \sqrt{x^2 + y^2 + z^2} - \sqrt{2}
   $$

2. 分别求偏导并代入点 `(1, 0, -1)`  

   - **`F_x`**  
     $$
     F_x = yz + \frac{1}{2\sqrt{x^2+y^2+z^2}} \cdot 2x = yz + \frac{x}{\sqrt{x^2+y^2+z^2}}
     $$
     代入 `x=1, y=0, z=-1`：
     $$
     F_x = 0 + \frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2}
     $$

   - **`F_y`**  
     $$
     F_y = xz + \frac{y}{\sqrt{x^2+y^2+z^2}}
     $$
     代入：`y=0` 消去第二项，`xz = 1·(-1) = -1`，所以 `F_y = -1`。

   - **`F_z`**  
     $$
     F_z = xy + \frac{z}{\sqrt{x^2+y^2+z^2}}
     $$
     代入：`xy = 0`，`z = -1`，`√(···) = √2`，所以：
     $$
     F_z = 0 + \frac{-1}{\sqrt{2}} = -\frac{\sqrt{2}}{2}
     $$

3. 套公式求偏导  

   - $$
     \frac{\partial z}{\partial x} = -\frac{F_x}{F_z} = -\frac{\frac{\sqrt{2}}{2}}{-\frac{\sqrt{2}}{2}} = 1
     $$
   - $$
     \frac{\partial z}{\partial y} = -\frac{F_y}{F_z} = -\frac{-1}{-\frac{\sqrt{2}}{2}} = -\frac{2}{\sqrt{2}} = -\sqrt{2}
     $$

4. 写出全微分  
   $$
   dz = \frac{\partial z}{\partial x} \, dx + \frac{\partial z}{\partial y} \, dy = dx - \sqrt{2} \, dy
   $$
