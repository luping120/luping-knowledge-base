## 二元隐函数定义 *Content-[00:00]

- 不是形如 \( z = f(x, y) \) 的二元函数，即由方程 \( F(x, y, z) = 0 \) 所确定的函数称为二元隐函数。
- 判断标准：函数表达式不是显式给出的 \( z \) 关于 \( x, y \) 的关系式。

## 二元隐函数一阶偏导求解方法 *Content-[00:50]

- 主要使用 **公式法** 求解一阶偏导。
- 公式：
  \[
  \frac{\partial z}{\partial x} = -\frac{F_x}{F_z},\quad \frac{\partial z}{\partial y} = -\frac{F_y}{F_z}
  \]
  其中 \( F(x, y, z) = 0 \) 是由方程移项得到的函数。

## 公式法步骤 *Content-[01:46]

1. **移项构造**：将方程整理为 \( F(x, y, z) = 0 \) 的形式（左边减右边，或右边减左边）。
2. **求偏导**：分别计算 \( F_x \)、\( F_y \)、\( F_z \)（对 \( x \)、\( y \)、\( z \) 求偏导，其余变量视为常数）。
3. **代入公式**：使用上述公式计算 \( \frac{\partial z}{\partial x} \) 和 \( \frac{\partial z}{\partial y} \)。

## 例题一：简单隐函数求偏导 *Content-[02:54]

**题目**：已知方程 \( z + e^z = xy \) 确定 \( z = z(x, y) \)，求 \( \frac{\partial z}{\partial x} \)、\( \frac{\partial z}{\partial y} \)。

**解**：
- 令 \( F(x, y, z) = z + e^z - xy = 0 \)
- 求偏导：
  \[
  F_x = -y,\quad F_y = -x,\quad F_z = 1 + e^z
  \]
- 套公式：
  \[
  \frac{\partial z}{\partial x} = -\frac{-y}{1+e^z} = \frac{y}{1+e^z},\quad \frac{\partial z}{\partial y} = -\frac{-x}{1+e^z} = \frac{x}{1+e^z}
  \]

## 例题二：含根号的隐函数求全微分 *Content-[05:26]

**题目**（2015年真题，4分）：方程 \( xyz + \sqrt{x^2 + y^2 + z^2} = \sqrt{2} \) 确定 \( z = z(x, y) \)，求在点 \( (1, 0, -1) \) 处的全微分 \( dz \)。

**解**：
- 令 \( F(x, y, z) = xyz + \sqrt{x^2 + y^2 + z^2} - \sqrt{2} = 0 \)
- 计算偏导（代入点 \( (1, 0, -1) \)）：
  - \( F_x = yz + \frac{x}{\sqrt{x^2+y^2+z^2}} \)，代入得 \( F_x = 0 + \frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2} \)
  - \( F_y = xz + \frac{y}{\sqrt{x^2+y^2+z^2}} \)，代入得 \( F_y = -1 + 0 = -1 \)
  - \( F_z = xy + \frac{z}{\sqrt{x^2+y^2+z^2}} \)，代入得 \( F_z = 0 + \frac{-1}{\sqrt{2}} = -\frac{\sqrt{2}}{2} \)
- 套公式：
  \[
  \frac{\partial z}{\partial x} = -\frac{F_x}{F_z} = -\frac{\frac{\sqrt{2}}{2}}{-\frac{\sqrt{2}}{2}} = 1,\quad \frac{\partial z}{\partial y} = -\frac{F_y}{F_z} = -\frac{-1}{-\frac{\sqrt{2}}{2}} = -\sqrt{2}
  \]
- 全微分：
  \[
  dz = \frac{\partial z}{\partial x} dx + \frac{\partial z}{\partial y} dy = 1 \cdot dx + (-\sqrt{2}) \cdot dy = dx - \sqrt{2}\,dy
  \]
