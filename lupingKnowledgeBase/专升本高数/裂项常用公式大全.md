## 基础分式型

这是最基础的模型，通项为分母是两个因式乘积的形式。

### 通用公式

$\frac{1}{n(n+k)} = \frac{1}{k} \left( \frac{1}{n} - \frac{1}{n+k} \right)$

### 常用特例

- $\frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$
- $\frac{1}{n(n+2)} = \frac{1}{2} \left( \frac{1}{n} - \frac{1}{n+2} \right)$
- $\frac{1}{(2n-1)(2n+1)} = \frac{1}{2} \left( \frac{1}{2n-1} - \frac{1}{2n+1} \right)$
- $\frac{1}{n(n+1)(n+2)} = \frac{1}{2} \left[ \frac{1}{n(n+1)} - \frac{1}{(n+1)(n+2)} \right]$

## 根式型

当分母含有根号且为两项之和时，利用平方差公式进行裂项。

### 通用公式

$\frac{1}{\sqrt{n} + \sqrt{n+k}} = \frac{1}{k} (\sqrt{n+k} - \sqrt{n})$

### 常用特例

- $\frac{1}{\sqrt{n} + \sqrt{n+1}} = \sqrt{n+1} - \sqrt{n}$
- $\frac{1}{\sqrt{2n-1} + \sqrt{2n+1}} = \frac{1}{2} (\sqrt{2n+1} - \sqrt{2n-1})$
- $\frac{1}{\sqrt{n} + \sqrt{n+k}} = \frac{\sqrt{n+k} - \sqrt{n}}{k}$

## 阶乘与组合数型

这类公式在处理涉及阶乘或排列组合的数列时非常有用。

### 阶乘相关

- $\frac{n}{(n+1)!} = \frac{1}{n!} - \frac{1}{(n+1)!}$
- $\frac{1}{n(n+1)(n+2)} = \frac{1}{2} \left[ \frac{1}{n(n+1)} - \frac{1}{(n+1)(n+2)} \right]$
- $\frac{1}{(n+1)(n+2)} = \frac{1}{n+1} - \frac{1}{n+2}$

### 组合数相关

利用组合数性质 $C_{n+1}^m = C_n^m + C_n^{m-1}$ 变形：

- $C_n^{m-1} = C_{n+1}^m - C_n^m$
- $\frac{C_n^m}{C_{n+1}^m} = \frac{n+1-m}{n+1}$

## 进阶/特殊型

这些公式在难题中偶尔会出现，属于"大招"。

### 对数型

利用对数的运算性质 $\ln a - \ln b = \ln(\frac{a}{b})$ 逆运算。

- $\ln \left( 1 + \frac{1}{n} \right) = \ln \left( \frac{n+1}{n} \right) = \ln(n+1) - \ln n$
- $\log_a \left( 1 + \frac{1}{n} \right) = \log_a (n+1) - \log_a n$

### 指数型

- $\frac{2^n}{(2^n-1)(2^{n+1}-1)} = \frac{1}{2^n-1} - \frac{1}{2^{n+1}-1}$
- $\frac{3^n}{(3^n-1)(3^{n+1}-1)} = \frac{1}{2} \left( \frac{1}{3^n-1} - \frac{1}{3^{n+1}-1} \right)$

### 三角函数型

- $\tan(a-b) = \frac{\tan a - \tan b}{1 + \tan a \tan b}$
- $\cot(a-b) = \frac{\cot a \cot b + 1}{\cot b - \cot a}$

## 记忆与使用技巧

### 系数确定方法

**看分母差值**：对于 $\frac{1}{A \cdot B}$ 的形式，裂项后一定要除以 $(B-A)$ 这个差值。

- 例如：$\frac{1}{3 \cdot 5} = \frac{1}{5-3}(\frac{1}{3} - \frac{1}{5}) = \frac{1}{2}(\frac{1}{3} - \frac{1}{5})$

### 消项规律

- 如果裂项是 $a_n - a_{n+1}$（相邻项），求和后通常剩 **首项** 和 **末项**
- 如果裂项是 $a_n - a_{n+2}$（隔一项），求和后通常剩 **前两项** 和 **后两项**
- 一般地，如果裂项是 $a_n - a_{n+k}$，求和后通常剩 **前k项** 和 **后k项**

### 验证方法

如果你记不清系数是 $\frac{1}{2}$ 这是 $\frac{1}{3}$，直接把右边通分算一下，看是否等于左边即可。

### 通用步骤

1. **观察通项形式**：确定属于哪种类型
2. **选择合适公式**：根据分母特征选择公式
3. **确定系数**：注意差值的倒数
4. **验证正确性**：通分检验
5. **求和计算**：写出前几项和后几项，观察抵消规律

---

**提示**：裂项相消法的关键在于"消"，通过构造相邻项的差，使得中间项相互抵消，最终只留下有限项进行计算。
